# 设计讨论 #03 — 经济系统

> 状态: **草案 v4.2**（奶红回调原值 + 满杯百香果机制修正 + 好感经验曲线）
> 日期: 2026-05-30
> 模型: **街机投币（投N币出N绳 → 纯概率负期望 → 虚假操控感）**

---

## 核心理念：虚假斯金纳箱

**拍拍拍是一个虚假的斯金纳箱系统**。

玩家以为自己的连击影响了结果，实际上——

```
投币瞬间 → 概率已经骰好了 → 结果已定
拍拍拍阶段 → 纯表演 → 无论玩家怎么拍，结果不变
```

设计目的：

1. **制造"手感"幻觉** — 连击反馈让玩家以为自己在施加影响
2. **归因偏差** — 抓到鬼 → "我拍得好！" 没抓到 → "下次拍快点"
3. **心流维持** — 即使概率负期望，操控感让玩家持续投入

### 表演层实现（已确认）

| 花活 | 状态 | 说明 |
|------|------|------|
| 拍得快→进度条跑得快 | ❌ 不做 | 动画固定 |
| "差一格"心理 | ❌ 不做 | — |
| **连击音效 + 屏幕震动** | ✅ 做 | 强化参与感，唯一的反馈通道 |

## 1. 抓鬼经济（主循环 — 负期望）

### 投入模型

```
每回合面对 1 只鬼（按固定顺序轮番出场）
投入 = 玩家自选 1~5 铜钱（每投1铜钱 = 多1条链）
每条链独立判定 catchP（同一只鬼）
```

### 鬼种参数（已确认 v2 — 最高锚定 0.6）

| 鬼 | 抓获概率 (catchP) | 赔率范围 | 中值赔率 (midOdds) | 单链 EV | 说明 |
|----|-------------------|---------|-------------------|---------|------|
| 牛头 | 20% | ×1~×3 | 2 | **0.400** | 最常见，低回报 |
| 马面 | 10% | ×4~×6 | 5 | **0.500** | 中等 |
| 黑无常 | 5% | ×7~×12 | 9.5 | **0.475** | 少见 |
| 白无常 | 3% | ×13~×20 | 16.5 | **0.495** | 稀有 |
| 刑天 | 1.2% | ×20~×50 | 35 | **0.420** | 极稀有大奖 |

> **单链 EV = catchP × midOdds**
> 最高单链 EV = 马面 0.500 < 0.6 ✅ 锚定满足

### 基础期望值

```
轮转平均 base EV = (0.400 + 0.500 + 0.475 + 0.495 + 0.420) / 5
                 = 0.458 / coin

含义：不买任何 buff，每投1铜钱，期望回收 0.458 铜钱
      即 ~54% 的庄家优势
```

### 判定流程（虚假斯金纳箱实现）

```
1. 当前回合出场的鬼已确定（轮番）
2. 玩家投 N 铜钱 → 生成 N 条链
3. 每条链骰 catchP，结果已定
4. 进入"拍拍拍"阶段（纯视觉表演）：
   - 音效反馈 + 屏幕震动
5. 揭晓：成功 → +赔率×1铜钱 / 失败 → 链断
```

### 爆奖控制

```lua
-- 最近10局 ROI 动态调节
if ROI > 0.8 then 高级鬼(白无常/刑天)出场概率 ×0.5
if ROI < 0.4 then 低级鬼(牛头/马面)出场概率 ×1.5
```

## 2. 打工/黄金矿工（正期望 — 回血）

**主玩法横向甩链钩鬼，打工纵向下钩捞东西**。

| 好感等级 | 每局打工收入 | 相当于主玩法 |

| Lv.2 | 10 铜钱 | = 2波满投 |
| Lv.3 | 20 铜钱 | = 4波满投 |

- **冷却机制**：打工 = 时间成本

## 3. 孟婆好感系统

### 核心机制

**"请孟婆喝一杯"** 有两种效果：

1. **盲盒** — 随机获得 #1~#8 中一种 buff
2. **+好感经验** — 积累升级孟婆好感等级（5级，影响打工收入）

### 好感升级经验曲线

| 等级 | 升级所需 | 累计投入 | 说明 |
|------|---------|---------|------|
| Lv.1 → Lv.2 | 3 杯 | 3 杯 (60 铜钱) | 快速体验升级 |
| Lv.2 → Lv.3 | 8 杯 | 11 杯 (220 铜钱) | 中期目标 |
| Lv.3 → Lv.4 | 15 杯 | 26 杯 (520 铜钱) | 长线投资 |
| Lv.4 → Lv.5 | 25 杯 | 51 杯 (1020 铜钱) | 终极目标 |

> 每杯"请孟婆喝一杯" = 20 铜钱
> 满级投入 1020 铜钱，打工收入从 5→50（+45/局），约 23 局打工回本

**曲线特征**：
- 前期快（3杯即升级，正反馈明确）
- 后期陡（25杯才满级，但每级收入跳跃大，值得追）
- 总投入 1020 铜钱 vs 满级后每局 +45 铜钱 = 长线必投

## 4. 孟婆奶茶（10种 — 完整数值表）

### 命名体系

- **奶红系×4** = 加概率（catchP 加算百分点）
- **奶绿系×4** = 加收益（赔率加算，整数）
- **特调系×2** = 特殊机制

### 基础计算公式

```
奶红 buff (+X pp catchP):
  新 EV(鬼g) = (catchP_g + X%) × midOdds_g
  平均 ΔEV = X% × avg(midOdds) = X% × 13.6

奶绿 buff (+A 赔率加算):
  新 EV(鬼g) = catchP_g × (midOdds_g + A)
  平均 ΔEV = A × avg(catchP) = A × 0.0784

组合（奶红+奶绿同时生效）:
  新 EV(鬼g) = (catchP_g + X%) × (midOdds_g + A)
  含交叉项 X%×A，组合收益 > 各自之和
```

### C. 特调系 — 特殊机制（2种）

#### #9 满杯百香果 — 锁定鬼种

| 属性 | 值 |
|------|-----|
| **效果** | 接下来 3 回合，**全部鬼替换为同一种**（随机决定哪种） |
| **持续** | 3回合 |
| **定价** | 10 铜钱 |

**机制说明**：

```
喝下百香果 → 系统随机骰一种鬼(uniform 1/5)
→ 接下来3回合全部出这种鬼（不轮番）
→ 玩家看到锁定结果后，可决定是否追加奶红
```

**单独使用**：

```
期望 EV = 各鬼 base EV 的均值 = 0.458（不变！）
但方差极大：锁到刑天 = 3回合全是×35赔率机会
            锁到牛头 = 3回合全是×2赔率...
```

**核心价值：信息确定性 + 组合精准狙击**

玩家喝百香果后看到锁定了哪只鬼，再决定是否追加奶红：

| 锁定鬼种 | base EV | +阿萨姆(+1.5pp) | +红糖(+2.5pp) | +万里(+4pp) |
|---------|---------|---------------|-------------|------------|
| 牛头 | 0.400 | 0.430 | 0.450 | 0.480 |
| 马面 | 0.500 | 0.575 | 0.625 | 0.700 |
| 黑无常 | 0.475 | 0.618 | 0.713 | 0.855 |
| 白无常 | 0.495 | 0.743 | 0.908 | 1.155 |
| **刑天** | 0.420 | **0.945** | **1.295** | **1.820** |

> **策略**：百香果锁到刑天/白无常 → 立刻追加奶红 → 3回合爆发
> 百香果锁到牛头 → 放弃追加，省钱等下一轮
>
> 这创造了**决策点**：百香果本身便宜(10铜钱)，风险低，但打开了高收益组合的窗口

#### #10 杨枝甘露 — 超高概率钩

| 属性 | 值 |
|------|-----|
| **效果** | 本回合 5 条链中的 **1 条** 变为 ×10 catchP（超级钩） |
| **Cap** | catchP 上限 100%（超过视为必中） |
| **持续** | 1回合（单次使用） |
| **定价** | 25 铜钱 |

**各鬼计算**（5链 = 1 超级钩 + 4 普通钩）：

| 鬼 | 超级钩 catchP | 超级钩 EV | 普通钩 EV×4 | 5链总 EV | 每币 EV |
|----|--------------|----------|------------|---------|---------|
| 牛头 | 200%→100% | 1×2=**2.00** | 4×0.40=1.60 | 3.60 | **0.720** |
| 马面 | 100%→100% | 1×5=**5.00** | 4×0.50=2.00 | 7.00 | **1.400** |
| 黑无常 | 50% | 0.50×9.5=**4.75** | 4×0.475=1.90 | 6.65 | **1.330** |
| 白无常 | 30% | 0.30×16.5=**4.95** | 4×0.495=1.98 | 6.93 | **1.386** |
| 刑天 | 12% | 0.12×35=**4.20** | 4×0.42=1.68 | 5.88 | **1.176** |

```
轮转平均 EV（杨枝甘露，5链满投）:
  = (0.720 + 1.400 + 1.330 + 1.386 + 1.176) / 5
  = 1.202 / coin

有效 EV（含茶钱）= 5×1.202 / (5+25) = 6.01/30 = 0.200
```

> **杨枝甘露 = 赌狗路径**：单回合超高爆发，但含茶钱后是最负期望的选择。
> 适合在高赔率鬼出场时"赌一把"——命中白无常 ×20 → 单链回报 20 铜钱。
>
> **与百香果组合**：先用百香果锁到高赔率鬼，再用杨枝甘露一击必杀。

### G. buff 通用规则

- **叠加**：奶红+奶绿可同时生效（交叉项生效）；同类覆盖不叠加
- **特调**：与奶红/奶绿可同时生效
- **新买覆盖旧**：同类 buff 刷新时长+效果
- **百香果+杨枝甘露**：可以组合（锁定高赔率鬼 → 超级钩一击）

## 6. 防崩溃机制

| 风险 | 对策 |
