# 设计讨论 #03 — 经济系统

> 状态: **草案 v4.0**（用户确认：修正鬼参数 + 10种奶茶完整数值表 + 定价）
> 日期: 2026-05-29
> 模型: **街机投币（投N币出N绳 → 纯概率负期望 → 虚假操控感）**

---

## 核心理念：虚假斯金纳箱

**拍拍拍是一个虚假的斯金纳箱系统**。

玩家以为自己的连击影响了结果，实际上——

```
投币瞬间 → 概率已经骰好了 → 结果已定
拍拍拍阶段 → 纯表演 → 无论玩家怎么拍，结果不变
```

设计目的：

1. **制造"手感"幻觉** — 连击反馈让玩家以为自己在施加影响
2. **归因偏差** — 抓到鬼 → "我拍得好！" 没抓到 → "下次拍快点"
3. **心流维持** — 即使概率负期望，操控感让玩家持续投入

### 表演层实现（已确认）

| 花活 | 状态 | 说明 |
|------|------|------|
| 拍得快→进度条跑得快 | ❌ 不做 | 动画固定 |
| "差一格"心理 | ❌ 不做 | — |
| **连击音效 + 屏幕震动** | ✅ 做 | 强化参与感，唯一的反馈通道 |

## 1. 抓鬼经济（主循环 — 负期望）

### 投入模型

```
每回合面对 1 只鬼（按固定顺序轮番出场）
投入 = 玩家自选 1~5 铜钱（每投1铜钱 = 多1条链）
每条链独立判定 catchP（同一只鬼）
```

### 鬼种参数（已确认 v2 — 最高锚定 0.6）

| 鬼 | 抓获概率 (catchP) | 赔率范围 | 中值赔率 (midOdds) | 单链 EV | 说明 |
|----|-------------------|---------|-------------------|---------|------|
| 牛头 | 20% | ×1~×3 | 2.0 | **0.400** | 最常见，低回报 |
| 马面 | 10% | ×4~×6 | 5.0 | **0.500** | 中等 |
| 黑无常 | 5% | ×7~×12 | 9.5 | **0.475** | 少见 |
| 白无常 | 3% | ×13~×20 | 16.5 | **0.495** | 稀有 |
| 刑天 | 1.2% | ×20~×50 | 35.0 | **0.420** | 极稀有大奖 |

> **单链 EV = catchP × midOdds**
> 最高单链 EV = 马面 0.500 < 0.6 ✅ 锚定满足

### 基础期望值

```
轮转平均 base EV = (0.400 + 0.500 + 0.475 + 0.495 + 0.420) / 5
                 = 2.290 / 5
                 = 0.458 / coin

含义：不买任何 buff，每投1铜钱，期望回收 0.458 铜钱
      即 ~54% 的庄家优势
```

### 判定流程（虚假斯金纳箱实现）

```
1. 当前回合出场的鬼已确定（轮番）
2. 玩家投 N 铜钱 → 生成 N 条链
3. 💀 每条链骰 catchP，结果已定
4. 进入"拍拍拍"阶段（纯视觉表演）：
   - 音效反馈 + 屏幕震动
5. 揭晓：成功 → +赔率×1铜钱 / 失败 → 链断
```

### 爆奖控制

```lua
-- 最近10局 ROI 动态调节
if ROI > 0.8 then 高级鬼(白无常/刑天)出场概率 ×0.5
if ROI < 0.4 then 低级鬼(牛头/马面)出场概率 ×1.5
```

## 4. 孟婆奶茶（10种 — 完整数值表）

### 命名体系

- **奶红系×4** = 加概率（catchP 加算百分点）
- **奶绿系×4** = 加收益（赔率加算）
- **特调系×2** = 特殊机制

### 基础计算公式

```
奶红 buff (+Xpp catchP):
  新 EV(鬼g) = (catchP_g + X) × midOdds_g
  平均 ΔEV = X × avg(midOdds) = X × (2+5+9.5+16.5+35)/5 = X × 13.6

奶绿 buff (+A 赔率加算):
  新 EV(鬼g) = catchP_g × (midOdds_g + A)
  平均 ΔEV = A × avg(catchP) = A × (0.20+0.10+0.05+0.03+0.012)/5 = A × 0.0784

组合（奶红+奶绿同时生效）:
  新 EV(鬼g) = (catchP_g + X) × (midOdds_g + A)
  = base_EV_g + X×midOdds_g + A×catchP_g + X×A  (交叉项)
```

#### #10 杨枝甘露 — 超高概率钩

| 属性 | 值 |
|------|-----|
| **效果** | 本回合 5 条链中的 **1 条** 变为 ×10 catchP（超级钩） |
| **Cap** | catchP 上限 100%（超过视为必中） |
| **持续** | 1回合（单次使用） |
| **定价** | 25 铜钱 |

**各鬼计算**（5链 = 1 超级钩 + 4 普通钩）：

| 鬼 | 超级钩 catchP | 超级钩 EV | 普通钩 EV×4 | 5链总 EV | 每币 EV |
|----|--------------|----------|------------|---------|---------|
| 牛头 | 200%→100% | 1×2=**2.00** | 4×0.40=1.60 | 3.60 | **0.720** |
| 马面 | 100%→100% | 1×5=**5.00** | 4×0.50=2.00 | 7.00 | **1.400** |
| 黑无常 | 50% | 0.50×9.5=**4.75** | 4×0.475=1.90 | 6.65 | **1.330** |
| 白无常 | 30% | 0.30×16.5=**4.95** | 4×0.495=1.98 | 6.93 | **1.386** |
| 刑天 | 12% | 0.12×35=**4.20** | 4×0.42=1.68 | 5.88 | **1.176** |

```
轮转平均 EV（杨枝甘露，5链满投）:
  = (0.720 + 1.400 + 1.330 + 1.386 + 1.176) / 5
  = 6.012 / 5
  = 1.202 / coin

ΔEV vs base = 1.202 - 0.458 = +0.744
```

> **注意**：杨枝甘露只生效1回合。5链满投时总花费 = 25(茶)+5(投币) = 30 铜钱，总期望回收 = 5×1.202 = 6.01 铜钱。
> 有效 EV = 6.01/30 = 0.200（含茶钱后大幅负期望！但爆发感极强）
>
> **实际价值**：杨枝甘露适合在高赔率鬼出场时"赌一把"——如果对马面使用，5链 EV=7.0，有效 EV = 7/(5+25)=0.233；如果命中白无常 ×20，单链回报 20 铜钱。
>
> 与满杯百香果组合：先用百香果把高赔率鬼概率拉高，再用杨枝甘露一击必杀。

### D. 组合策略与有效 EV 表

**设计目标**：多次购买组合 → 有效 EV 提升到 0.8~0.9

**有效 EV 公式**：

```
有效 EV = (buff期间投币数 × buffed_EV) / (buff期间投币数 + 茶钱总花费)

假设: 每回合投满5链
```

#### 常见组合（5回合buff，每回合5链 = 25投币）

| 组合 | 茶钱 | buffed avg EV | 投币期望回收 | 有效 EV | 评价 |
|------|------|--------------|------------|---------|------|
| 无buff（裸玩） | 0 | 0.458 | 11.45 | **0.458** | 基线 |
| 阿萨姆(8) | 8 | 0.662 | 16.55 | **0.501** | 微提升 |
| 茉香(6) | 6 | 0.615 | 15.38 | **0.496** | 微提升 |
| 阿萨姆+茉香 | 14 | 0.818* | 20.45 | **0.524** | 还不够 |
| 红糖珍珠(15) | 15 | 0.798 | 19.95 | **0.499** | 中等 |
| 红糖+薄荷 | 27 | 1.110* | 27.75 | **0.534** | 越来越好 |
| **万里木兰(20)** | 20 | 1.002 | 15.03† | **0.501** | 本身>1但3回合 |
| **万里+伯牙** | 38 | 1.551* | 23.27† | **0.403** | 超贵但爆发 |

> \* 组合 EV 含交叉项：(catchP+X)×(midOdds+A) 中的 X×A
> † 3回合buff: 投币数=15

**等等，有效 EV 怎么越组合越低了？**

这是故意的设计！**奶茶的价值不在于"打平"或"赢钱"，而在于：**

1. **爆发感** — 单回合可能回收 ×50（红豆双皮奶 + 刑天命中）
2. **心理锚定** — "差一点就回本了"（负期望但比裸玩好）
3. **Buff 期间的中奖率体感飙升** — 即使期望是负的，频繁小奖让人兴奋

### E. 重新定义"0.8~0.9"目标

用户指示：*"通过宏观调控把期望提升到0.8~0.9甚至超出1一点点也没问题只是要多次购买并且运气在正常水平"*

**正确解读**：0.8~0.9 是 **buffed per-coin EV**（不含茶钱），不是有效 EV。

| 目标 buffed EV | 达成方式 |
|---------------|---------|
| 0.8 | 红糖珍珠(+2.5pp) 单独 → 0.798 ≈ 0.8 ✅ |
| 0.85 | 阿萨姆+薄荷 → 0.662+0.314-0.458=... 不对，需重算组合 |
| 0.9 | 红糖珍珠+满杯百香果(加权) → 0.910 ✅ |
| >1.0 | 万里木兰 单独 → 1.002 ✅ / 杨枝甘露 → 1.202 ✅ |

**修正后的组合 buffed EV 表**（不含茶钱，纯投币回报）：

| 组合方案 | 总茶钱 | buffed avg EV | 达标？ |
|---------|-------|--------------|--------|
| 阿萨姆奶红 (单) | 8 | 0.662 | ✗ |
| 红糖珍珠 (单) | 15 | **0.798** | ≈0.8 ✅ |
| 阿萨姆+茉香 | 14 | **0.818** | ✅ |
| 阿萨姆+薄荷 | 20 | **0.976** | ✅✅ |
| 红糖+百香果(加权) | 30 | **0.910** | ✅ |
| 红糖+薄荷 | 27 | **1.112** | >1 ✅✅ |
| 万里木兰 (单) | 20 | **1.002** | >1 ✅ |
| 红豆双皮奶 (单) | 35 | **1.274** | >>1 |

> 阿萨姆+茉香 组合详算：
> 各鬼 EV = (catchP+0.015)×(midOdds+2)
> 牛头: 0.215×4 = 0.860
> 马面: 0.115×7 = 0.805
> 黑无常: 0.065×11.5 = 0.748
> 白无常: 0.045×18.5 = 0.833
> 刑天: 0.027×37 = 0.999
> avg = (0.860+0.805+0.748+0.833+0.999)/5 = 4.245/5 = **0.849**
>
> (之前写0.818是近似，精确值0.849因含交叉项0.015×2=0.03)

**结论**：

- **入门路径**：阿萨姆+茉香 = 14铜钱 → buffed EV 0.85 ✅
- **标准路径**：红糖珍珠 = 15铜钱 → buffed EV 0.80 ✅
- **进阶路径**：红糖+百香果 = 30铜钱 → buffed EV 0.91 ✅
- **爆发路径**：万里木兰 = 20铜钱 → buffed EV 1.00（3回合，短而猛）
- **赌狗路径**：杨枝甘露 = 25铜钱 → 单回合 1.20（1回合超爆发）
