## 已定：打工 = 黄金矿工

| 项目 | 确认 |
|------|------|
| 玩法 | 黄金矿工（放钩子抓东西） |
| 场景 | 从孟婆店切换回游戏区（缓动过渡动画） |
| 入口 | 孟婆对话选"我来打工" |
| 奖励 | 赚货币（符？待确认） |
| 主题适配 | 钩子→锁链，金块→冥界宝物/游魂碎片 |

**和主玩法的关系**：主玩法是横向甩钩钩鬼，打工是纵向下钩捞东西。操作手感相似但方向不同，不会腻。

---

## 已定：展示机制 = 保持原代码风格

原套牛代码的展示方式：
- 抓到时：游戏区内 **浮动消息**（黑底圆角框 + 白字，1.5s 淡出）
- 大赢时：**"BIG WIN!" 弹性文字**（outElastic 缓动）
- 没有暂停、没有卡片、没有弹窗

转化到钟馗：
- 抓到鬼 → 浮动文字显示鬼的名字/一句话黑色幽默
- 大赢 → "镇鬼成功!" + 弹性特效文字
- 风格保持休闲、不打断节奏

## 已定：货币 = 铜钱（方孔钱）✅

| 对比 | 街机 | 钟馗版 |
|------|------|--------|
| 形态 | 圆形金属硬币 | **铜钱**（圆形方孔，金属质感） |
| 动作 | 投币 | 花铜钱开局 |
| 获取 | — | 抓鬼爆落 + 打工（黄金矿工）赚 |
| 花法 | 投1币=1局 | 花N铜钱=1局 + 孟婆店买道具 |
| 视觉 | 💰 | 古铜色方孔钱，弹跳掉落 |
| 音效 | 叮叮当当 | 同（金属碰撞音，复用原代码逻辑） |

### 为什么选铜钱？

1. **爆出来合理**：鬼是地下的东西，冥界流通铜钱，抓到鬼→鬼掉随身铜钱
2. **街机感完美映射**：视觉动画、音效、手感直接复用原代码 coin burst
3. **道士不冲突**：五帝钱本就是道士法器之一
4. **孟婆收铜钱**：冥界通用货币，商店自洽
5. **实现成本最低**：改贴图 + 音效微调就行，逻辑完全复用原套牛
