# 九宫格（9-Slice）边框纹饰使用指南

> **这是什么**: UrhoX UI 系统中使用九宫格纹理实现可拉伸边框的完整指南，含 UI 库和 NanoVG 两种方案。
> **原作者**: 南北绿豆
> **推荐度**: L1（直接可用的代码参考 + 素材规格说明）
> **适用场景**: UI 按钮/面板/卡片需要装饰性边框且要适应不同尺寸时

---

## 九宫格原理

将一张图片分为 9 区域：4 角不拉伸、4 边单向拉伸、中心双向拉伸。一张 256x256 边框可无损应用到任意大小组件。

## 方法一：UrhoX UI 内置（推荐）

```lua
UI.Button {
    text = "开始修炼",
    width = 200, height = 60,
    backgroundImage = "image/九宫格_普通按钮边框.png",
    backgroundFit = "sliced",
    backgroundSlice = {64, 64, 64, 64},  -- {上, 右, 下, 左}
    onClick = function(self) end,
}

UI.Panel {
    width = 400, height = 300,
    backgroundImage = "image/九宫格_普通按钮边框.png",
    backgroundFit = "sliced",
    backgroundSlice = {64, 64, 64, 64},
    padding = 20,
    children = { ... }
}
```

## backgroundSlice 参数

顺序同 CSS：`{上, 右, 下, 左}`（顺时针）。本套素材统一 64px。

**注意**: 组件宽高应 > slice 两倍（即 > 128px），否则角落重叠。

## 素材规格

- 原始完整边框：256×256 PNG
- 切割边距：统一 64px
- 切片目录：`image/九宫格切片/`（每种边框 9 片）
- 使用 UI 库内置九宫格时**直接用完整原图**即可
