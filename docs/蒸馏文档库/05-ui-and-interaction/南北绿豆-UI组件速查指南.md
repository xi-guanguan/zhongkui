# UI 组件速查指南

> **作者**: 南北绿豆 | **难度**: L1-实操 | **来源**: TapTap 论坛 2026-05

## 核心理念

> 你不需要记住每个组件的用法细节——那是 AI 的事。你只需要**知道有这个东西，记住它的名字**。

当你对 AI 说"帮我用 Drawer 做一个从右边滑出来的设置面板"，AI 就不会给你手搓一百行代码了。

---

## 八大分类速查表

### 一、布局组件（6 个）

界面的骨架，决定"东西放在哪里"。

| # | 组件 | 用途 |
|---|------|------|
| 1 | **Panel** | 万能容器。背景色/渐变/圆角/阴影 + Flexbox 布局。卡片、页面背景、分组区域 |
| 2 | **Label** | 文本标签。字号/颜色/粗体/描边/阴影/下划线/多行。标题、分数、倒计时 |
| 3 | **Divider** | 分割线。水平/垂直，实线/虚线/点线，可带标签文字 |
| 4 | **SimpleGrid** | 网格布局。等宽列自动排列。背包格子、技能栏、卡牌手牌 |
| 5 | **SafeAreaView** | 安全区域。自动检测刘海/圆角/底部条，手机上架必备 |
| 6 | **ScrollView** | 滚动视图。惯性滚动/弹性回弹/滚动条/吸附。长列表、设置页、商店 |

### 二、输入控件（9 个）

接收用户操作，交互核心。

| # | 组件 | 用途 |
|---|------|------|
| 7 | **Button** | 按钮。多种变体，悬停/按压状态。游戏中出现频率最高 |
| 8 | **TextField** | 文本输入框。光标/选中/占位/密码模式/长度限制 |
| 9 | **Checkbox** | 复选框。设置开关、多选筛选、协议勾选 |
| 10 | **Toggle** | 开关。iOS 风格滑动开关，适合"开/关"二元设置 |
| 11 | **Slider** | 滑块。音量/亮度/灵敏度/属性加点 |
| 12 | **Stepper** | 步骤指示器。新手引导/角色创建/任务进度 |
| 13 | **Rating** | 评分。星级/半星/自定义图标。关卡评分、满意度 |
| 14 | **FileUpload** | 文件上传（模拟交互）。头像上传、地图导入 |
| 15 | **ButtonGroup** | 按钮组。工具栏操作组、视图切换、筛选条件 |

### 三、选择控件（7 个）

从一组选项中挑选值。

| # | 组件 | 用途 |
|---|------|------|
| 16 | **Dropdown** | 下拉选择。语言/难度/服务器/排序切换 |
| 17 | **DatePicker** | 日期选择器。活动日期、签到日历 |
| 18 | **TimePicker** | 时间选择器。闹钟、定时任务 |
| 19 | **ColorPicker** | 颜色选择器。角色染色、画笔颜色 |
| 20 | **Calendar** | 日历。签到系统、活动日程 |
| 21 | **Menu** | 菜单/右键菜单。嵌套子菜单/图标/快捷键 |
| 22 | **Tree** | 树形视图。文件目录、技能分类、组织架构 |

### 四、展示控件（9 个）

信息呈现，数据可视化。

| # | 组件 | 用途 |
|---|------|------|
| 23 | **Card** | 卡片。头部/内容/底部区域。物品详情、商品卡 |
| 24 | **Badge** | 徽标。未读数/红点/在线状态（支持动画） |
| 25 | **Chip** | 标签。技能标签/属性词条/筛选标签，可选中可删除 |
| 26 | **Avatar** | 头像。圆形/方形，图片/首字母/状态指示 |
| 27 | **Alert** | 提示横幅。信息/成功/警告/错误消息条 |
| 28 | **Tooltip** | 悬浮提示。按钮说明/属性详情/快捷键提示 |
| 29 | **Skeleton** | 骨架屏。加载占位动画（脉冲/波浪） |
| 30 | **ProgressBar** | 进度条。经验条/血量条/下载进度（支持动画） |
| 31 | **RichText** | 富文本。HTML 标签/颜色/粗体/物品引用 |

### 五、导航控件（3 个）

页面/区域切换。

| # | 组件 | 用途 |
|---|------|------|
| 32 | **Tabs** | 标签页。商店分类（武器/防具/道具）、面板切换 |
| 33 | **Breadcrumb** | 面包屑。层级路径导航 |
| 34 | **Pagination** | 分页。排行榜翻页、搜索结果 |

### 六、容器控件（6 个）

组织和展示结构化内容。

| # | 组件 | 用途 |
|---|------|------|
| 35 | **Accordion** | 折叠面板。FAQ、分组设置、技能说明 |
| 36 | **Carousel** | 轮播。活动横幅、新闻公告、教程幻灯片 |
| 37 | **Timeline** | 时间轴。版本日志、任务进度链、成长历程 |
| 38 | **Table** | 数据表格。排行榜、属性对比、战斗统计 |
| 39 | **List** | 列表。设置选项、联系人、任务清单 |
| 40 | **VirtualList** | 虚拟列表。万级数据流畅滚动（对象池+视图回收） |

### 七、浮层控件（4 个）

覆盖在页面之上的临时交互。

| # | 组件 | 用途 |
|---|------|------|
| 41 | **Modal** | 模态弹窗。确认对话框、物品详情、成就解锁 |
| 42 | **Drawer** | 侧边抽屉。设置面板、物品侧栏、导航菜单 |
| 43 | **Popover** | 气泡浮层。操作确认、快捷操作面板 |
| 44 | **Toast** | 临时通知。操作反馈、获得物品、成就达成 |

### 八、游戏专用组件（6 个）

游戏场景专门设计的高级组件。

| # | 组件 | 用途 |
|---|------|------|
| 45 | **ChatWindow** | 聊天窗口。MMO 聊天、NPC 对话、系统日志 |
| 46 | **SkillTree** | 技能树。可缩放平移，三种节点状态，自动连线 |
| 47 | **ItemSlot** | 物品格子。背包格/装备栏/快捷栏，支持拖放 |
| 48 | **DragDropContext** | 拖放上下文。管理拖拽状态，配合 ItemSlot |
| 49 | **ItemTooltip** | 物品提示。品质着色/属性数值/悬浮详情窗 |
| 50 | **InventoryManager** | 背包管理器。纯数据模型，物品增删查移逻辑 |

## 使用方式

1. 根据需求在上表中**找到组件名字**
2. 告诉 AI："帮我用 [组件名] 做 [具体功能]"
3. AI 会直接生成对应的代码实现

**收藏本文，开发时翻一翻，找到名字，告诉 AI——就这么简单。**
