# 横向标签栏的滚动实现

> **作者**: 游玩 | **级别**: L1-实操 | **标签**: `ScrollView` `横向滚动` `标签栏` `Flexbox`

---

## 三层结构

```
ScrollView (scrollX=true, scrollY=false, showScrollbar=false)
  └─ Panel (flexDirection="row", 不设宽度)
       ├─ Item1 (height="100%", aspectRatio=1, flexShrink=0)
       ├─ Item2 (...)
       ├─ Item3 (...)
       └─ ...更多标签
```

---

## 关键代码

```lua
local tabBar = UI.ScrollView {
    width = "100%",
    height = 60,
    scrollX = true,
    scrollY = false,
    showScrollbar = false,
    children = {
        UI.Panel {
            flexDirection = "row",
            -- ⚠️ 不要设置 width！让内容撑开
            height = "100%",
            children = tabs
        }
    }
}
```

每个标签项：

```lua
UI.Panel {
    height = "100%",
    aspectRatio = 1,      -- 正方形标签
    flexShrink = 0,       -- ⚠️ 关键：不压缩！
    justifyContent = "center",
    alignItems = "center",
    children = { UI.Label { text = "Tab" } }
}
```

---

## 两个必须

| 规则 | 原因 |
|------|------|
| 容器**不设 width** | 让子元素总宽度自然溢出，触发滚动 |
| 子项 **flexShrink = 0** | 防止 Yoga 把子项压缩到容器内，破坏滚动 |

---

## 常见错误

```lua
-- ❌ 容器设了固定宽度 → 子项被压缩，无法滚动
UI.Panel { width = "100%", flexDirection = "row", ... }

-- ❌ 子项没设 flexShrink=0 → 被压缩到容器内
UI.Panel { height = "100%", aspectRatio = 1, ... }  -- 默认 flexShrink=1 会压缩
```
