# 更省积分、更精准还原布局的 UI 制作方案

> **作者**: 织匠 | **级别**: L2-参考 | **来源**: TapTap 开发心得（精华帖）

---

## 核心思路

**DeepSeek V4 识图** → 反推样式代码 → 在嗒啦啦中一次对话还原。

省积分的关键：把"反复调整"的过程放在免费的 DeepSeek 中完成，只给嗒啦啦最终版代码。

---

## 完整工作流

### 准备阶段
准备一张 UI 布局图（自绘或 AI 生成）

### 第一步：DeepSeek 识图 → 描述内容
1. 登录网页版 DeepSeek，选择**识图模式**，打开**深度思考**
2. 上传 UI 布局图
3. 输入："描述图片内容"

> 这一步不省略！不加这步可能导致后续样式代码无法运行或布局有出入。

### 第二步：反推样式代码
输入："根据图片反推该界面的样式代码"

- DeepSeek 通常能给出可直接运行的代码
- 点击运行按钮预览效果
- 如果无法运行，告诉它"给一个可以直接运行的代码版本"

### 第三步：在 DeepSeek 中调整
- 简单修改：直接文字描述
- 复杂修改：截图标注需要改的部分 + 文字辅助说明
- 如果给出代码片段，让它"整合为完整代码"

### 第四步：导入嗒啦啦
复制最终版样式代码，在嗒啦啦中输入：
```
根据以下样式代码，修改[界面名称]的UI布局：
[粘贴代码]
```

**通常只需这一次对话**即可完整还原。

### 后续微调
- 小修改：用嗒啦啦的"截图插入对话"功能，圈出修改区域
- 大修改（界面重做）：回到 DeepSeek 重新做

---

## 优势对比

| 方式 | 积分消耗 | 还原度 | 调整难度 |
|------|----------|--------|----------|
| 直接和嗒啦啦反复对话 | 高（每轮几百积分） | 不稳定 | 高（经常抽风） |
| **DeepSeek调试 → 嗒啦啦一次还原** | **极低（1次对话）** | **高** | **低（DeepSeek免费）** |

---

## 社区补充方案

| 来源 | 方案 |
|------|------|
| Dash | 同样的图直接发给嗒啦啦识图也行，但调整消耗积分多 |
| 长宁Aring | 用 Axure 生成 HTML → 提取 CSS 参数 → 给嗒啦啦按参数制作 |
| 零殇 | 嗒啦啦识图可以，但提示词禁止"网页代码"会出问题 |

**本质**：都是获取 CSS 参数交给嗒啦啦执行，DeepSeek 方案免费且灵活。

---

## 注意事项

- 如何精确替换美术素材仍在探索中
- 嗒啦啦对"网页代码"有提示词限制，需绕过
- 大幅修改建议回 DeepSeek 重做而非在嗒啦啦迭代
