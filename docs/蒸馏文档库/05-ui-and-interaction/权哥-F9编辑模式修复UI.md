# F9 编辑模式修复 UI 的焚诀

> **作者**: 权哥 | **级别**: L3-注意 | **标签**: `F9` `UI调试` `Inspector` `工作流`

---

## F9 做什么

F9 进入 UI 编辑/检查模式（类似浏览器 DevTools），可以：
- 悬停高亮任意 UI 元素
- 点击选中查看属性
- 拖拽调整位置
- 多选批量操作

---

## 操作流程

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 按 F9 | 进入编辑模式，游戏暂停交互 |
| 2 | 鼠标悬停 | 高亮元素边界框 |
| 3 | 点击/框选 | 选中元素，底部显示属性 |
| 4 | 拖拽 | 调整位置（实时预览） |
| 5 | 再按 F9 | 退出编辑模式 |

---

## 配合 AI 修 UI 的工作流

```
F9 调整 → 复制属性值 → 粘贴给 AI 对话 → AI 改代码 → 构建生效
```

**关键**：F9 调整是临时预览，刷新后会恢复。要永久生效必须改代码后重新构建。
