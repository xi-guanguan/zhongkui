# 如何提升你游戏中的 UI 美术水平

> **作者**: N3 | **级别**: L1-实操 | **标签**: `UI美术` `字体` `图标` `布局` `九宫格`

---

## 一、字体选择

| 字体类型 | 适用场景 | 示例 |
|----------|----------|------|
| 黑体 | 标题、按钮、正式文本 | 思源黑体、MiSans |
| 宋体 | 古风、文学、叙事 | 思源宋体 |
| 楷体 | 手写感、轻松氛围 | — |
| 圆体 | 儿童向、萌系 | — |
| 科技体 | 赛博朋克、机甲 | — |

**免费字体资源**：100font.com

**上传技巧**：字体文件加 `.png` 后缀上传平台，使用时去掉后缀即可。

## 二、图标设计

**核心原则：扁平化 + 简化**

- 去掉多余细节，保留核心轮廓
- 统一图标风格（线条粗细、圆角、配色）
- 小尺寸下（32×32）细节越少越清晰

---

## 三、文字可读性

### 描边 + 背景色双保险

```
文字颜色：白色/亮色
描边：深色 1-2px（确保任何背景下可读）
关键数值背景：半透明深色色块
```

**关键数值**（血量、分数等）用加粗 + 放大 + 对比色突出。

---

## 四、布局原则

### 左右分栏布局

```
┌──────────┬──────────────┐
│  导航栏   │   内容区域    │
│  (固定宽) │  (自适应宽)   │
└──────────┴──────────────┘
```

- 重要信息放左上（视线起点）
- 操作按钮放右下（拇指热区）
- 留足间距，不要挤满

---

## 五、卡片 UI 设计

```
┌─────────────────┐
│  [图标]  标题    │
│                 │
│  关键数值（大号）│
│  次要信息（小号）│
│                 │
│  [操作按钮]     │
└─────────────────┘
```

- 数值用大号字体 + 强调色
- 次要信息灰色/小号
- 按钮区与信息区分离

## 六、倾斜文字技巧

用 `nvgSkewX` 或 `transform` 让文字微倾（5°~15°），增加动感：
- 适合：伤害数字、连击提示、技能名
- 不适合：正文、菜单项

---

## 七、九宫格切图（Nine-Patch）

将 UI 背景图切成 9 块，四角固定、边缘拉伸、中间平铺：

```
┌───┬─────┬───┐
│ 1 │  2  │ 3 │  ← 四角不拉伸
├───┼─────┼───┤
│ 4 │  5  │ 6 │  ← 上下边横向拉伸
├───┼─────┼───┤
│ 7 │  8  │ 9 │  ← 左右边纵向拉伸
└───┴─────┴───┘     中间双向拉伸
```

**优势**：一张小图适配任意尺寸面板，不会变形。
