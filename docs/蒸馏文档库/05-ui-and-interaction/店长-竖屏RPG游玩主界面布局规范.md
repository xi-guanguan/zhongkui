# 竖屏 RPG 游玩主界面布局规范

> **这是什么**：一份竖屏卡牌/RPG 游戏的主界面完整布局设计文档，涵盖顶部信息栏、底部导航栏、战斗场景三大区域，精确到每个 UI 元素的坐标、尺寸、字号、颜色、描边参数。
>
> **原作者**：店长
> **推荐度**：L1（坐标级精度，拿来就能照搬实现）
> **适用场景**：制作竖屏（1080×2400）卡牌/RPG/放置类游戏的主界面时，可作为完整的布局参考或直接复用

---

## 1. 设计基准

| 项目 | 值 |
|------|-----|
| 设计分辨率 | 1080 × 2400（竖屏） |
| 坐标原点 | 左上角 |
| 坐标含义 | 所有 X/Y 均为元素**中心点**位置 |
| 尺寸单位 | 像素（px） |

> 所有素材坐标和尺寸以此分辨率为准，代码中应将其写死为设计分辨率常量，运行时通过缩放适配。

## 5. 全局样式规范

### 5.1 文本描边

所有游戏内文本统一使用**纯黑描边**，仅描边粗细有差异：

| 字号范围 | 描边大小 | 使用场景 |
|---------|---------|---------|
| 28-33 | 4 | 常规信息文本（名称、数值、战力） |
| 40 | 4 | 区域标签（"敌人"、关卡名、按钮文本） |
| 60 | 6 | 导航栏选中标签标题 |

### 5.2 颜色速查

| 用途 | 颜色 |
|------|------|
| 通用文本 | 纯白 #FFFFFF |
| 战力文本 | #f7fe77（亮黄） |
| 后退按钮叠加 | #5ac7c8 60%（青色） |
| 前进按钮叠加 | #d94949 60%（红色） |
| 背景半透明 | 纯黑 40%~60% |

### 5.3 进度条通用规则

所有进度条（经验/血量/攻击）遵循相同模式：
- 背景图 + 填充图 两层结构
- 填充图在背景图内部，**四周内边距 4px**
- 填充图宽度按百分比裁剪表示进度
