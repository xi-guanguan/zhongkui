# 清除存档功能说明

> 作者: 店长 | 级别: L2-参考 | 编号: #158

---

## 概述

多人游戏中完整的"清除存档"功能实现：UI 按钮、协议定义、服务端处理、客户端全量重置流程。

| 文件 | 职责 |

## 1. UI 入口（DebugPanel）

深红色按钮，文案"清除存档(危险)"，点击后二次确认再发送请求。

---

## 2. 协议定义

```lua
ACTION_TYPES.GM_RESET_SAVE = "gm_reset_save"
```

客户端发送：
```lua
Client.SendToServer(NetworkEvents.GM_ACTION, {
    action = ACTION_TYPES.GM_RESET_SAVE
})
```

---

## 3. 服务端处理链

### GMHandler（路由）
```lua
function GMHandler.OnGMAction(uid, data)
    if data.action == ACTION_TYPES.GM_RESET_SAVE then
        GMService.ResetSave(uid)
    end
end
```

### GMService.ResetSave（核心）
```lua
function GMService.ResetSave(uid)
    -- 遍历 CharacterSchema.Fields，每个字段重置为默认值
    for fieldKey, def in pairs(CharacterSchema.Fields) do
        local default = CharacterSchema.Defaults[fieldKey]
        if def.type == "scalar" then
            PDM.SetStat(uid, def.pdmKey, default or 0)
        elseif def.type == "json" then
            PDM.SetStruct(uid, def.pdmKey, default or {})
        end
    end

    -- 清除 global_profile 中的 servers 和 serverProgress
    PDM.SetStruct(uid, "GlobalProfile", { servers = {}, serverProgress = {} })

    -- 即时存盘（不等心跳）
    PDM.SavePlayer(uid)

    -- 通知客户端重置
    ServerSync.SendToClient(uid, NetworkEvents.GM_RESET_COMPLETE, { ok = true })
end
```

## 4. 客户端全量重置（12步）

```lua
function Standalone.requestResetToStartScreen()
    -- 1. 停止所有定时器
    TimerManager.StopAll()
    -- 2. 清理战斗状态
    BattleManager.Cleanup()
    -- 3. 清理 UI 栈
    ScreenManager.ClearAll()
    -- 4. 重置事件总线
    EventBus.Reset()
    -- 5. 断开网络连接
    NetworkClient.Disconnect()
    -- 6. 清理 PlayerStore 缓存
    PlayerStore.Cleanup()
    -- 7. 清理 SyncClient
    SyncClient.Reset()
    -- 8. 重置音频状态
    AudioManager.StopAll()
    -- 9. 清理场景节点
    SceneManager.Cleanup()
    -- 10. 重置一次性标志
    Client.resetForNewSession()
    -- 11. 重置局部状态变量
    -- (各模块自有的 local 状态)
    -- 12. 跳转到起始画面
    ScreenManager.Switch("StartScreen")
end
```

## 5. 一次性标志重置

```lua
function Client.resetForNewSession()
    hasShownWelcome_ = false
    hasLoadedInventory_ = false
    tutorialStep_ = 0
    -- ... 所有模块的 "已初始化" 标记
end
```

## 关键避雷

| 问题 | 解决 |
|------|------|
| 清除后客户端缓存残留 | 必须执行完整 12 步重置 |
| 只清服务端不清客户端 | 客户端收到 GM_RESET_COMPLETE 后执行本地重置 |
| 一次性标志未重置 | 新会话某些流程不触发（如教学） |
| 心跳未到时断线 | ResetSave 后立即调 SavePlayer，不依赖心跳 |
