# 星火素材库模型动画加载技巧：从搜索到播放动画

> **这是什么**：在 UrhoX（TapTap 制造/星火编辑器）中加载星火素材库的 3D 模型并播放动画的完整流程。解决素材库模型常见的"无贴图"和"动画播不出来"两大问题。
>
> **原作者**：心如梦
> **推荐度**：L3（流程清晰可用，但文章较早期，部分 API 细节可能随引擎更新有变化；uuid:// 加载方式是核心，这个不会变）
> **适用场景**：需要从 TapTap 星火素材库加载 3D 模型（怪物、角色等）到游戏中，并播放行走/攻击等动画

---

## 一、流程总览

```
search_3d_resource 搜索模型
       ↓
检查返回的 type 字段 → 优先选 type="prefab"
       ↓
GetResource("XMLFile", "uuid://...") → LoadXML（自动带贴图）
       ↓
在节点树中查找 AnimatedModel 组件
       ↓
AutoScale 缩放到目标高度
       ↓
AnimationController 放在 AnimatedModel 同一节点
       ↓
用 animation_data 中的 UUID 播放动画
```

---

## 二、加载模型

### 2.1 选择资源类型

> ⚠️ **优先选 `type="prefab"` 的资源**——这是唯一能自动带贴图的方式。其他类型可能加载后模型是白的。

### 2.2 加载 Prefab

```lua
local PREFAB_UUID = "G4wooYs70lk6oNbiOh_VzRkC"  -- 替换为实际 UUID
local modelNode = parentNode:CreateChild("ModelNode")

-- 加载 prefab XML（自动包含材质贴图）
local prefabFile = cache:GetResource("XMLFile", "uuid://" .. PREFAB_UUID)
if prefabFile then
    modelNode:LoadXML(prefabFile:GetRoot())
end
```

### 2.3 查找 AnimatedModel

**AnimatedModel 不一定在根节点上，可能在子节点**。需要遍历查找：

```lua
local animModel = modelNode:GetComponent("AnimatedModel")
if not animModel then
    for i = 0, modelNode:GetNumChildren(false) - 1 do
        local child = modelNode:GetChild(i)
        animModel = child:GetComponent("AnimatedModel")
        if animModel then break end
    end
end
```

## 三、播放动画

> ⚠️ **关键规则**：AnimationController 必须创建在 AnimatedModel **同一节点**上，不能放在父节点或其他节点。

```lua
local animModelNode = animModel.node
local animCtrl = animModelNode:GetOrCreateComponent("AnimationController")

-- 播放动画（UUID 从 animation_data.animations_data 中获取）
animCtrl:PlayExclusive("uuid://EhkjydJVXUCN2VNluMLKYZaa", 0, true, 0.2)
-- 参数：动画资源路径, layer, looped, fadeTime
```

---

## 四、自动缩放

素材库模型尺寸不统一，用 `boundingBox` 自动缩放到目标高度：

```lua
local size = animModel.boundingBox.size
local targetHeight = 1.5  -- 目标高度（米）

if size.y > 0.01 then
    local s = targetHeight / size.y
    modelNode:SetScale(Vector3(s, s, s))
end
```

## 五、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 模型加载后是白色/无贴图 | 用了非 prefab 类型的资源 | 搜索时优先选 `type="prefab"` |
| 动画播不出来 | AnimationController 不在 AnimatedModel 同节点 | `GetOrCreateComponent` 必须在 `animModel.node` 上调用 |
| 模型巨大或极小 | 素材库模型尺寸不统一 | 用 `boundingBox.size` 自动缩放 |
| 找不到 AnimatedModel | 组件在子节点而非根节点 | 遍历子节点查找 |
