# 程序化 3D 建模系统工具链

## 这是什么

三层程序化建模工具栈（MeshBuilder 底层 + MHL 高层 + Profiles 截面库），用纯 Lua 代码生成 3D 模型，无需外部建模软件。覆盖 11 个核心工作流（环形建模/拓扑连接/定向拉伸/挖洞/焊接合并等），含颅部建模 7 步法和完整踩坑清单。

- **原作者**: 山姆
- **推荐度**: L2（参考学习）—— 工具链完整但从另一项目移植，当前项目尚未引入实际代码文件
- **适用场景**: 需要纯代码程序化生成 3D 角色/道具/有机形体的项目，尤其是低多边形风格

---

## 三层架构

```
┌──────────────────────────────────────────────┐
│  Part Builders — 每部件一个 Build 函数，组合下层 │
├──────────────────────────────────────────────┤
│  PartMeshes — CreateTube/CreatePBRMaterial    │
│  直接操作 CustomGeometry + Material            │
├──────────────┬─────────────┬─────────────────┤
│ MeshBuilder  │ MHL         │ Profiles        │
│ 底层:顶点/面 │ 高层:样条管  │ 截面库:圆/椭/盾 │
│ 环/变换/拓扑 │ 旋转体/基本体│ 缩放/旋转/插值  │
└──────────────┴─────────────┴─────────────────┘
```

核心数据结构：`mesh = { verts={{x,y,z},...}, faces={{v1,v2,v3},...}, uvs={{u,v},...} }`

## MeshBuilder (MB) — 底层 API 精选

| 类别 | 核心 API |
|------|---------|
| 基础 | `create()`, `addVert()`, `addFace()`, `faceNormal()`, `select(pred)` |
| 变换 | `translateMesh()`, `scaleMesh()`, `rotateMeshX/Y/Z()` |
| 环形 | `makeRing()` Z轴向, `makeRingY()` Y轴向, `fromRings()` 多环连管 |
| 拓扑 | `extrudeFacesDirected()`, `deleteFaces()`, `weldVertices()`, `mergeAtVertices()` |
| 合并 | `merge()`, `mirrorX()`, `symmetrize()` |
| 变形 | `bend()`, `twist()`, `taper()` — 低多边形可用; `subdivide()`/`noiseDisplace()` — 低多边形禁用 |
| 提交 | `commit()` flat无UV, `commitTextured()` flat+UV, `commitSmooth()`, `commitTexturedSmooth()` |

## MHL — 高层生成器

### splineTube（最常用）

沿路径生成变截面管体，**不需要轴修正**：

```lua
local mesh = MHL.splineTube({
    path = {{0,0,0}, {0,-0.02,0.015}, ...},  -- 路径控制点
    radiiFn = function(t) return r end,        -- 半径函数
    segments = 12, steps = 20,
    profile = Pro.circle(12),                  -- 截面形状
    capStart = true, capEnd = true,
})
```

### lathe（旋转体）

沿 Y 轴生成，**需要 rotateMeshX(90) 修正**：

```lua
local mesh = MHL.lathe({ profile = {{r,y},...}, segments = 16 })
MB.rotateMeshX(mesh, 90)  -- Y→Z 修正
```

### 其他基本体

sphere/hemisphere/cone/capsule — 内部用 lathe，均需 `rotateMeshX(90)`。
torus — 独立环实现，不需要修正。

## 截面库 Profiles

| 类型 | 函数 | 用途 |
|------|------|------|
| 基础 | `circle(seg)`, `ellipse(rx,ry,seg)` | 通用管体 |
| 有机 | `shield(w,h,seg)`, `teardrop(w,h,seg)`, `star(oR,iR,pts)` | 颅部/液囊/棘刺 |
| 结构 | `roundedRect(w,h,r,cs)`, `dShape(w,h,seg)`, `hexagon(r)` | 甲片/甲壳/节段 |
| 变换 | `scale()`, `rotate()`, `translate()`, `lerp()`, `resample()` | 截面插值过渡 |

`resample(profile, targetCount)` — 弧长重采样，让不同段数截面可插值。

## 坐标系与轴修正 🔴

| 生成器 | 需要 rotateMeshX(90)? | 有偏移时坐标交换? |
|--------|---------------------|-----------------|
| splineTube | ❌ 不需要 | N/A |
| segmentedChain | ❌ 不需要 | N/A |
| lathe/sphere/cone/capsule/hemisphere | ✅ 需要 | ✅ 传入 (tx, tz, -ty) |
| torus | ❌ 不需要 | ❌ |

**修正数学**：`rotateMeshX(90°)` → `(X,Y,Z)→(X,-Z,Y)`。要让最终位置 = `(tx,ty,tz)`，传入 = `(tx, tz, -ty)`。

## 11 个核心工作流

| # | 工作流 | 关键 API | 适用场景 |
|---|--------|---------|---------|
| 1 | 环形建模 | makeRing + fromRingsIndexed | 躯干/肢体/角 |
| 2 | 拓扑连接 | fromRingsIndexed + addVert/addFace | 牙齿从颌骨长出 |
| 3 | 定向拉伸 | select + extrudeFacesDirected | 角/刺/触手 |
| 4 | 挖洞 | select + deleteFaces | 眼眶/鼻孔 |
| 5 | 焊接合并 | merge + weldVertices | 消除接缝 |
| 6 | 拓扑合并 | mergeAtVertices | 部件焊接到主体 |
| 7 | 边缘环挤出 | extrudeEdgeLoop | 裙边/底座 |
| 8 | 拉普拉斯平滑 | smoothVertices | 柔化有机形体 |
| 9 | 对称建模 | symmetrize | 只建半边自动镜像 |
| 10 | 异构环放样 | loftBetween | 不同段数截面过渡 |
| 11 | 表面投影 | projectOntoSurface | 装饰物贴合曲面 |

## 颅部建模 7 步法

核心方法论：「七分真（原型动物解剖）+ 三分幻（夸张/幻想元素）」

### 特征分类

| 类 | 表达方式 | 位移量占 halfH |
|----|---------|--------------|
| A类 顶点塑形 | 直接修改 ring 顶点 | 主隆起 60~80%, 次隆起 50~65%, 外扩 40~55%, 凹陷 20~35%, 纹理 10~18% |
| B类 独立几何 | 独立子节点+不同材质 | ≤15 个子节点 |

**黄金法则**：位移量太小 = 看不见 = 白做。宁可偏大再调小。

**B类定位补偿**：`附件Y = centerY + halfH + 最大A类位移 - 嵌入余量(0.002~0.005)`

## 踩坑清单

| # | 踩坑 | 正确做法 |
|---|------|---------|
| 1 | 对 splineTube 做 rotateMeshX(90) | splineTube 不需要轴修正 |
| 2 | 对 torus 做 rotateMeshX(90) | torus 不走 lathe，不需要修正 |
| 3 | lathe 有偏移不做坐标交换 | 传入 (x, z, -y) |
| 4 | Builder 里旋转 rootNode | 在几何层面修正方向 |
| 5 | 材质局部变量被 GC | 用 KeepMat 保持引用 |
| 6 | deleteFaces/weldVertices 忘赋值 | 必须 `mesh = MB.deleteFaces(mesh,…)` |
| 7 | 低多边形下用 subdivide/noiseDisplace | 直接用，不细分不加噪声 |
| 8 | A类位移量太小 | 至少 30%，主特征 50%+ |
| 9 | 小附件用主体截面(16点) | 小附件用 6 点圆环 |
| 10 | merge 后以为有拓扑连接 | 用 mergeAtVertices 共享顶点 |
