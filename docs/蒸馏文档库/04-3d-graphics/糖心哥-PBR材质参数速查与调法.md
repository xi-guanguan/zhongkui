# PBR 材质参数速查与调法

> **这是什么**：PBR 三核心参数（颜色/金属度/粗糙度）的完整速查表 + 25 种常见材质参考值 + 风格策略，消灭 3D 游戏"塑料感"。
>
> **原作者**：糖心哥
> **推荐度**：L2（零基础友好，参数值可直接查表使用，但部分高级用法如贴图材质需另查引擎文档）
> **适用场景**：调整 3D 模型材质质感、搭建场景材质方案、排查"塑料感"问题时

---

## 一、三核心参数

| 参数 | 范围 | 说明 |
|------|------|------|
| **颜色** (Color) | RGB 各 0.0～1.0 | 物体是什么颜色 |
| **金属度** (Metalness) | **0 或 1**（不用中间值） | 导电=1（金属），不导电=0（非金属） |
| **粗糙度** (Roughness) | 0.0～1.0 | 0=镜面光滑，1=完全哑光 |

> **金属度只用 0 和 1**，中间值会产生不自然的效果。
> **粗糙度**是最影响质感的参数，差 0.1 就能从"塑料"变"真实"。

### 粗糙度直觉对照

| 粗糙度 | 效果 | 类比 |
|--------|------|------|
| 0.05 | 镜面 | 镜子、平静水面 |
| 0.1 | 高光锐利 | 抛光金属、釉面陶瓷 |
| 0.2 | 明显光泽 | 打蜡的车、珠宝 |
| 0.3 | 柔和光泽 | 丝绸、新塑料 |
| 0.4 | 中等光泽 | 打蜡木头、旧塑料 |
| 0.5 | 微光泽 | 皮革、磨砂金属 |
| 0.6 | 略有光 | 普通木头 |
| 0.7 | 基本哑光 | 粗木头、旧皮革 |
| 0.8 | 哑光 | 石头、砖块、混凝土 |
| 0.9 | 几乎无光 | 粉笔、干泥土 |
| 1.0 | 完全哑光 | 最粗糙表面 |

## 三、透明材质

透明度由颜色的第四个值 **Alpha** 控制（0.0=全透明，1.0=不透明）。

| 材质 | 颜色 RGBA | 金属度 | 粗糙度 | 效果 |
|------|----------|--------|--------|------|
| 玻璃 | (0.7, 0.9, 1.0, **0.3**) | 0 | 0.05 | 透明+清晰反光 |
| 磨砂玻璃 | (0.9, 0.9, 0.9, **0.4**) | 0 | 0.5 | 透但模糊 |
| 水晶/宝石 | (0.5, 0.2, 0.7, **0.5**) | 0.05 | 0.08 | 通透有光泽 |
| 冰块 | (0.8, 0.9, 1.0, **0.35**) | 0 | 0.1 | 通透冰冷感 |

> 透明度 < 0.1 肉眼几乎看不见，玻璃建议 0.3，水晶建议 0.5。

## 四、发光材质（Emissive）

自发光颜色值可以**超过 1.0**，数值越大越亮：

| 亮度级别 | 发光值示例 | 效果 |
|---------|-----------|------|
| 普通 | (1.0, 0.5, 0.2) | 温暖微光 |
| 中等 | (2.0, 1.0, 0.4) | 明显发光 |
| 强烈 | (3.0, 1.2, 0.0) | 像小太阳 |

> 超过 5.0 会过曝泛白。1.0～3.0 一般够用。

### 常见发光效果

| 效果 | 颜色 | 金属度 | 粗糙度 | 发光色 |
|------|------|--------|--------|--------|
| 火焰/岩浆 | (1.0, 0.6, 0.1) | 0 | 0.5 | (3.0, 1.2, 0.0) 强橙 |
| 魔法水晶 | (0.3, 0.5, 1.0) α0.5 | 0 | 0.1 | (0.5, 1.0, 2.0) 冷蓝 |
| 霓虹灯 | (1.0, 0.2, 0.5) | 0 | 0.2 | (2.5, 0.5, 1.2) 亮粉 |
| 金色拾取物 | (1.0, 0.84, 0.0) | 1.0 | 0.2 | (0.5, 0.4, 0.0) 柔金 |

## 五、光照预设

材质再好，灯光不对效果也出不来：

| 预设 | 色调 | 最佳搭配 |
|------|------|---------|
| 白天 | 明亮偏暖白 | 金属材质（高光明显） |
| 黄昏 | 暖橙 | 石头/木头（粗糙材质更自然） |
| 夜晚 | 冷蓝偏暗 | 发光材质（环境暗发光才突出） |

> 材质怎么调都不好看 → 先换灯光试试。

## 六、搭配原则

### 对比原则

| 原则 | 说明 | 好的搭配 |
|------|------|---------|
| 粗糙 vs 光滑 | 画面层次感 | 光滑金属剑 + 粗糙皮革剑柄 |
| 冷色 vs 暖色 | 避免色调单一 | 暖色木质家具 + 冷色金属装饰 |
| 发光 vs 不发光 | 突出重点 | 暗色地牢 + 发光魔法宝箱 |

> **发光少用才有效果**。满屏发光没重点。发光用于：可拾取物品、危险区域、魔法效果。

## 七、风格策略

| 风格 | 金属度 | 粗糙度 | 颜色 | 发光 |
|------|--------|--------|------|------|
| **写实** | 严格 0/1 | 参照真实物品 | 偏灰偏暗 | 极少 |
| **卡通** | 全给 0 | 统一 0.4～0.6 | 纯色高饱和 | 可选 |
| **暗黑/恐怖** | 按材料 | 整体偏高 | 深色冷色调 | 少量诡异色（红/绿） |
| **赛博朋克** | 大量 1 | 0.3～0.5 | 深灰/暗蓝灰 | 大量霓虹（粉/蓝/紫） |
| **奇幻/中世纪** | 武器 1，建筑 0 | 建筑高，武器中 | 自然色 | 魔法道具蓝/紫/绿 |

## 八、常见坑

| 坑 | 原因 | 解决 |
|----|------|------|
| 所有物体都像塑料 | 粗糙度都一样 | 不同材质的粗糙度要有明显差异 |
| 非金属看起来怪 | 金属度给了 0.5 | 非金属=0，金属=1，不用中间值 |
| 发光没效果 | 场景太亮 | 降低环境光，或发光值调到 2.0+ |
| 颜色对但不真实 | 粗糙度不对 | 颜色管"是什么颜色"，粗糙度管"质感" |
| 纯红纯蓝很假 | 现实没有纯色 | 写实风格让颜色"脏"一点（混灰） |
| 透明物体看不见 | Alpha 太低 | 玻璃 ≥ 0.3，水晶 ≥ 0.5 |

## 九、水面材质

水面用引擎内置预设，不需手动调参：

```
「在场景中加一个水面，用引擎自带的水面材质。水面颜色偏蓝。」
```

可进一步调色：深绿色=沼泽、浅蓝色=热带海洋、深蓝色=深海。

## 十、提示词模板

| 模板 | 格式 |
|------|------|
| 单物体 | 「给 ___（物体）设材质。颜色 ___，___（金属/非金属），表面 ___（光滑/粗糙）」 |
| 金属 | 「给 ___（物体）___（金/银/铜/铁）金属材质，表面 ___（抛光/磨砂/锈迹）」 |
| 透明 | 「给 ___（物体）___（颜色）玻璃/水晶材质，透明度 ___%，表面 ___」 |
| 发光 | 「给 ___（物体）加自发光，发 ___（颜色）光，亮度 ___（微弱/中等/强烈）」 |
| 整体场景 | 「统一设材质：A 用 ___，B 用 ___，C 用 ___。整体偏 ___（风格）」 |
| 微调 | 「___（物体）材质不对，现在看起来 ___，想要 ___，帮我调」 |
| 类比现实 | 「给 ___（物体）做成像 ___（现实物品）的感觉」 |
