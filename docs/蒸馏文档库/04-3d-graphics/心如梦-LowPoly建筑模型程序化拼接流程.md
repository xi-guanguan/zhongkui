## 7 步标准流程

### Step 1：确定建筑定位

明确四个参数：
1. 建筑名称和功能
2. 格子大小（1×1 / 2×2 / 3×3）
3. 设计风格（田园/工业/军事/商业）
4. 核心特征元素（3-5 个）

### Step 2：规划布局

1. 画出四象限草图
2. 确定主建筑位置和尺寸
3. 规划附属元素分布
4. 检查边界约束

### Step 3：定义材质

```lua
-- 1. 主材质（墙壁/主体）
local mainMat = Helpers.CreatePBRMaterial(Helpers.RGB(...), metallic, roughness)
-- 2. 次要材质（屋顶/门/框架）
local roofMat = ...
local doorMat = ...
local frameMat = ...
-- 3. 装饰材质（金属/发光）
local metalMat = ...
local glowMat = ...  -- 如需发光效果
```

材质分三类：主材质（墙体）、次要材质（屋顶/门/框架）、装饰材质（金属/发光）。

### Step 4：构建主体

**从下往上构建**：
1. 地基/平台
2. 主建筑墙体
3. 屋顶
4. 门窗

### Step 5：添加特征元素

根据建筑类型添加辨识度元素：

| 建筑类型 | 特色元素示例 |
|---------|------------|
| 伐木场 | 圆木堆、斧头、木桩 |
| 采石场 | 石块堆、起重架、矿车 |
| 矿场 | 矿洞、轨道、发光矿脉 |

### Step 6：细节装饰

增加生活感：
- 工具（镐/斧/锤）
- 容器（箱子/木桶）
- 标志（旗帜/招牌/符号）

### Step 7：检查清单

- [ ] 所有元素在格子范围内
- [ ] 高度层次分明
- [ ] 材质配色协调
- [ ] 有特色识别元素
- [ ] 主体开启 `castShadows = true`
