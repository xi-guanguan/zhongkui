# 监狱建筑师风格角色建模方法论

> **这是什么**: 使用简洁几何体（球/胶囊/方块）+ 配件差异化 + 节点动画构建风格化角色的完整方法论。
> **原作者**: 山姆
> **推荐度**: L2（设计方法论参考，适用于低多边形风格化游戏）
> **适用场景**: 程序化角色生成；监狱建筑师/Rimworld 风格的角色系统设计

---

## 设计理念

- 基础体用简单几何形状（球、胶囊、方块），追求"故意简洁"
- 角色差异通过**比例、颜色、配件组合**体现，不通过顶点塑形
- **没有手臂连接** —— 浮空手持物
- 保留脚，用于走路动画可读性
- 所有动画 = 节点 position/rotation 补间，不需要骨骼

## 禁止事项

- 禁止环形截面堆叠成管状
- 禁止顶点塑形扣解剖细节
- 禁止骨骼蒙皮动画

## 角色节点层级

```
CharacterRoot
├── Body          ← 胶囊体（服装色）
├── Head          ← 球体（肤色）
│   ├── LeftEye / RightEye
│   ├── Hair / Hat / Ear_L/R / FacialHair
├── Hand_L / Hand_R (浮空)
│   └── HeldItem
├── Foot_L / Foot_R
└── Accessories   ← 围裙/背包等
```

## 配件系统

| 类型 | 挂点 | 种族相关 |
|------|------|---------|
| 发型 | Head | 分性别/种族 |
| 尖耳 | Head 两侧 | 精灵/地精 |
| 帽子 | Head 顶部 | 职业相关 |
| 围裙/腰带/背包 | Body | 通用装饰 |

## 个体差异化（10+ 轴）

通用轴：身高、体宽、头身比、肤色、头形、眼睛样式、发型、发色、体型变体、服装色

种族专属轴：
- **矮人**: 胡子样式/长度、鼻头红度
- **精灵**: 耳长/角度、头部拉长度
- **地精**: 耳大小/下垂角、鼻子大小、獠牙

## 动画方案

| 动作 | 节点操作 |
|------|---------|
| 走路 | Foot_L/R 交替 Z 偏移 + Body 微 Y 浮动 |
| 站立 | Body 微 Y 呼吸浮动 + Head 偶尔微转 |
| 拿东西 | Hand 移向目标 → HeldItem 挂载 |
| 情绪 | Head 上方冒符号 + Body 抖动/跳跃 |
