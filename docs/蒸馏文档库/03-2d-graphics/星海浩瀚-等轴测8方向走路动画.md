# 等轴测 8 方向走路动画

> **这是什么**: 等轴测游戏中角色 8 方向朝向判定与走路动画渲染的完整实现方案。
> **原作者**: 星海浩瀚
> **推荐度**: L2（概念参考 + 伪代码，原文为 C++ 风格，需转写为 Lua）
> **适用场景**: 等轴测 RPG / SLG 中角色需要面向移动方向播放走路动画

---

## 核心思路

1. 定义 8 方向枚举：N / NE / E / SE / S / SW / W / NW
2. 移动时根据 `atan2(dy, dx)` 计算屏幕空间角度
3. 按 45° 扇区划分到 8 方向
4. 根据方向选择精灵帧 / 渲染姿态

## 方向判定（atan2 扇区划分）

```
        -90° (N)
          |
-135°(NW) | -45°(NE)
          |
±180°(W)--+-- 0°(E)
          |
 135°(SW) |  45°(SE)
          |
         90° (S)
```

```lua
local angle = math.atan(dy, dx)  -- Lua 5.4: math.atan 两参数
if     angle < -math.pi*7/8 then dir = DIR_W
elseif angle < -math.pi*5/8 then dir = DIR_NW
elseif angle < -math.pi*3/8 then dir = DIR_N
elseif angle < -math.pi*1/8 then dir = DIR_NE
elseif angle <  math.pi*1/8 then dir = DIR_E
elseif angle <  math.pi*3/8 then dir = DIR_SE
elseif angle <  math.pi*5/8 then dir = DIR_S
elseif angle <  math.pi*7/8 then dir = DIR_SW
else                              dir = DIR_W
end
```

## 走路动画

- `walkFrame += dt * animSpeed` 每帧累加
- 用 `sin(walkFrame)` 做上下浮动 / 脚步摆动
- 到达目标时 `walkFrame = 0`（回归站立）

## 4 方向精灵 + 镜像复用

如果只有 4 方向精灵，可用镜像扩展到 8 方向：

| 8 方向 | 使用精灵 | 是否镜像 |
|--------|---------|---------|
| N / NE / NW | back | NW 镜像 |
| S / SE / SW | front | SW 镜像 |
| E | side | 正常 |
| W | side | 水平翻转 |
