# 光影渲染框架：灯光类型、阴影配置与环境光雾效

> **这是什么**：UrhoX 引擎 3D 场景光影渲染的完整配置指南。涵盖 LightGroup 一键预设、三种灯光类型（方向光/点光/锥光）的创建与参数、阴影质量调节、Zone 环境光与雾效、Renderer 全局渲染参数，以及阴影常见问题排查表。
>
> **原作者**：第八维
> **推荐度**：L2（结构清晰、参数完整，适合 lowpoly/地牢风格场景；实际效果需根据项目美术风格微调参数）
> **适用场景**：需要给 3D 场景添加光照/阴影、配置日夜/黄昏光照环境、解决阴影闪烁/条纹等问题

## 一、快速开始：LightGroup 预设

一键配置场景光照（含 Zone + 方向光 + 阴影），适合快速搭建：

```lua
local lgFile = cache:GetResource("XMLFile", "LightGroup/Daytime.xml")
local lg = scene_:CreateChild("LightGroup")
lg:LoadXML(lgFile:GetRoot())
```

| 预设文件 | 场景效果 |
|---------|---------|
| `LightGroup/Daytime.xml` | 白天 |
| `LightGroup/Dusk.xml` | 黄昏 |
| `LightGroup/Night.xml` | 夜晚 |

> 预设够用就不要手动配，减少出错。需要微调时再手动覆盖个别参数。

## 二、三种灯光类型

### 2.1 方向光（太阳/月亮）

亮度单位：**lux**（勒克斯）。模拟无限远光源，所有物体受到相同方向的平行光。

```lua
local lightNode = scene_:CreateChild("Sun")
lightNode.direction = Vector3(0.6, -1.0, 0.8)

local light = lightNode:CreateComponent("Light")
light.lightType = LIGHT_DIRECTIONAL
light.color = Color(1.0, 0.95, 0.8)
light.brightness = 1.0              -- lux
light.specularIntensity = 0.5
light.castShadows = true
```

### 2.2 点光（灯泡/火把）

亮度单位：**candela**（坎德拉）。从一个点向四周辐射，亮度随距离衰减。

```lua
local pNode = scene_:CreateChild("PointLight")
pNode.position = Vector3(0, 3, 0)

local light = pNode:CreateComponent("Light")
light.lightType = LIGHT_POINT
light.color = Color(1.0, 0.8, 0.5)
light.brightness = 100.0            -- candela
light.range = 10.0                  -- 影响范围（米）
light.castShadows = true
```

### 2.3 锥光（手电/聚光灯）

亮度单位：**candela**。锥形光束，有明确的照射方向和角度。

```lua
local sNode = scene_:CreateChild("SpotLight")
sNode.position = Vector3(0, 5, 0)
sNode.direction = Vector3(0, -1, 0)

local light = sNode:CreateComponent("Light")
light.lightType = LIGHT_SPOT
light.brightness = 200.0            -- candela
light.range = 15.0                  -- 影响范围（米）
light.fov = 45.0                    -- 锥角（度）
light.castShadows = true
```

### 亮度单位速查

| 灯光类型 | 亮度单位 | 典型值 |
|---------|---------|--------|
| 方向光 | lux | 0.5 ~ 2.0 |
| 点光 | candela | 50 ~ 500 |
| 锥光 | candela | 100 ~ 1000 |

## 三、阴影配置

### 3.1 全局阴影设置（Renderer）

```lua
renderer.drawShadows = true
renderer.shadowMapSize = 2048                      -- 512 / 1024 / 2048 / 4096
renderer.shadowQuality = SHADOWQUALITY_PCF_24BIT   -- 推荐
renderer.shadowSoftness = 2.0                      -- 0 = 硬阴影
```

**阴影质量枚举**（从低到高）：

| 枚举值 | 说明 |
|--------|------|
| `SHADOWQUALITY_SIMPLE_16BIT` | 基础 16 位 |
| `SHADOWQUALITY_SIMPLE_24BIT` | 基础 24 位 |
| `SHADOWQUALITY_PCF_16BIT` | PCF 软阴影 16 位 |
| `SHADOWQUALITY_PCF_24BIT` | **推荐**，PCF 软阴影 24 位 |
| `SHADOWQUALITY_VSM` | 方差阴影贴图 |
| `SHADOWQUALITY_BLUR_VSM` | 模糊方差阴影贴图 |

### 3.2 单灯阴影参数

```lua
-- 偏移（修复条纹 / 彼得潘问题）
light.shadowBias = BiasParameters(0.00025, 0.5, 0.5)
-- BiasParameters(constantBias, slopeScaledBias, normalOffset)

-- 级联阴影（仅方向光有效）
light.shadowCascade = CascadeParameters(10.0, 30.0, 80.0, 0.0, 0.8)
-- CascadeParameters(split1, split2, split3, split4, fadeStart)

-- 聚焦（提高阴影贴图有效分辨率）
light.shadowFocus = FocusParameters(true, true, true, 0.5, 1.0)
-- FocusParameters(focus, nonUniform, autoSize, quantize, minView)

-- 其他
light.shadowIntensity = 0.0    -- 0 = 全黑阴影, 1 = 无阴影
light.shadowResolution = 1.0   -- 倍率 0.25 ~ 1.0
```

### 3.3 阴影问题排查

| 现象 | 原因 | 解决方法 |
|------|------|---------|
| 条纹 / 摩尔纹 | constantBias 太小 | 增大 `BiasParameters` 的 constantBias |
| 彼得潘（物体悬空） | constantBias 太大 | 减小 constantBias，增大 normalOffset |
| 远处阴影模糊 | 级联分配不合理 | 调整 `CascadeParameters` 的 split 值 |
| 阴影闪烁 | 阴影贴图分辨率低 | 增大 `shadowMapSize` 或启用 `shadowFocus` |

## 四、Zone 环境光与雾效

```lua
-- 获取默认 Zone（推荐方式）
local zone = renderer:GetDefaultZone()

-- 环境光（均匀）
zone.ambientColor = Color(0.3, 0.3, 0.4)

-- 渐变环境光（天顶到地面，更真实）
zone.ambientGradient = true
zone.ambientStartColor = Color(0.5, 0.6, 0.8)   -- 天顶色
zone.ambientEndColor = Color(0.2, 0.2, 0.15)     -- 地面色

-- 距离雾
zone.fogColor = Color(0.6, 0.7, 0.8)
zone.fogStart = 50.0     -- 起始距离（米）
zone.fogEnd = 300.0      -- 完全覆盖距离（米）

-- 高度雾
zone.heightFog = true
zone.fogHeight = 0.0          -- 雾的基准高度
zone.fogHeightScale = 0.1     -- 高度衰减速度
```

## 五、Renderer 全局参数速查

```lua
renderer.HDRRendering = true               -- HDR 渲染
renderer.specularLighting = true           -- 高光反射
renderer.textureAnisotropy = 8             -- 各向异性等级 1/2/4/8/16
renderer.textureFilterMode = FILTER_ANISOTROPIC  -- 纹理过滤
renderer.dynamicInstancing = true          -- 动态合批
renderer.maxShadowMaps = 4                 -- 最大阴影贴图数
renderer.reuseShadowMaps = true            -- 复用阴影贴图
```

## 六、完整示例：白天户外场景

```lua
function CreateLighting()
    -- 太阳方向光
    local sunNode = scene_:CreateChild("Sun")
    sunNode.direction = Vector3(0.5, -1.0, 0.7)

    local sun = sunNode:CreateComponent("Light")
    sun.lightType = LIGHT_DIRECTIONAL
    sun.color = Color(1.0, 0.95, 0.85)
    sun.brightness = 1.2
    sun.castShadows = true
    sun.shadowBias = BiasParameters(0.00025, 0.5, 0.5)
    sun.shadowCascade = CascadeParameters(10.0, 30.0, 80.0, 0.0, 0.8)
    sun.shadowFocus = FocusParameters(true, true, true, 0.5, 1.0)

    -- Zone 环境光 + 雾
    local zone = renderer:GetDefaultZone()
    zone.ambientColor = Color(0.35, 0.35, 0.4)
    zone.fogColor = Color(0.7, 0.75, 0.85)
    zone.fogStart = 80.0
    zone.fogEnd = 400.0

    -- 全局阴影
    renderer.drawShadows = true
    renderer.shadowMapSize = 2048
    renderer.shadowQuality = SHADOWQUALITY_PCF_24BIT
    renderer.shadowSoftness = 1.5
end
```
