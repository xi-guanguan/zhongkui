# 精灵图代码动画参数参考

## 这是什么

从实际项目的 `Render.lua` / `Config.lua` 中提取的完整精灵动画参数手册，覆盖 3 个玩家角色 + 10 种怪物的 SPRITE_DEFS、攻击钟形缩放、受击三件套（抖动/缩放/闪白）、死亡动画时间线、阴影系数、LOD 分级等全部数值。

- **原作者**: 店长
- **推荐度**: L2（参考学习）—— 参数体系完整，但绑定特定项目的精灵资源，需根据自身项目调整
- **适用场景**: NanoVG 精灵图游戏的角色动画系统设计，受击/死亡/行走等动画参数调参参考

---

## 精灵图定义 (SPRITE_DEFS)

### 玩家角色（3 套）

每套 5 个动画：Idle/Attacking/Hurt/Dying/Walking

| 动画 | 典型帧数 | 列×行 |
|------|---------|-------|
| Idle | 12帧 | 4×3 |
| Attacking | 12~15帧 | 4×3 / 5×3 |
| Hurt | 12帧 | 4×3 |
| Dying | 15帧 | 5×3 |
| Walking | 18帧 | 6×3 |

### 怪物（10 种）

骷髅系(3)/兽人系(2)/哥布林系(2)/幽魂系(2)/独眼巨人(1)。
幽魂系 Moving 帧为 12（非18），文件名为 `Moving_spritesheet.png`（非 Walking）。

## 核心动画参数

### 攻击钟形缩放

攻击过程中角色按钟形曲线放大再缩回，模拟蓄力释放感：

```lua
-- t = attackTimer / atkDuration  (0→1)
local remapped = (t - peakRatio) / max(peakRatio, 1 - peakRatio)
local bell = (1 + cos(clamp(remapped,-1,1) * pi)) * 0.5
local scaleFactor = 1.0 + 0.1 * bell  -- 1.0 → 1.1 → 1.0
-- 帧间平滑: lerp(lastScale, scaleFactor, 0.3)
```

| 参数 | 值 |
|------|-----|
| 峰值帧 | 第 7 帧 |
| 最大缩放 | 1.1x (110%) |
| 平滑因子 | 0.3 |
| Y轴补偿 | `-(charH - baseH) * 0.4`（脚底不动） |

### 受击三件套

| 效果 | 持续 | 关键参数 |
|------|------|---------|
| **抖动** | 0.12s | 普通3px/暴击6px, 频率19Hz, 线性衰减, 仅X轴 |
| **缩放弹跳** | 0.15s | 普通1.15x/暴击1.3x, 正弦半波曲线 |
| **闪白** | 0.15s | 加法混合×2次叠加, 最大60%透明度 |

### 死亡动画时间线

```
0% ──────── 40% ──────── 100%
│ 仅淡出+旋转 │ 淡出+缩小+旋转 │
│ alpha:1→0.6 │ alpha:0.6→0    │
│ scale:1.0   │ scale:1.0→0.3  │
│ rot:0→10°   │ rot:10°→25°    │
```

旋转方向随机（`dyingRotDir` = 1 或 -1）。阴影同步淡出。离场后阴影额外 0.5s 淡出。

## Idle 摆动与玩家参数

| 参数 | 值 |
|------|-----|
| 摆动公式 | `sin(gameTime * 5.4) * 5.0` 度 |
| 旋转支点 | 底部 40% 处 |
| 角色缩放 | 0.75 × idleSprite.fh |
| Y 位置 | 屏幕高度 77% |
| 攻击帧率 | 24 fps |
| 攻击发射帧 | 第 10 帧 |

## 特殊状态透明度

| 状态 | 公式 | 范围 |
|------|------|------|
| 灵体无敌 | `0.3 + 0.15*sin(t*6)` | 0.15~0.45 |
| 灵体现形 | `0.3 + 0.7*(fadeIn/0.5)` | 0.3→1.0 (0.5s) |
| 隐身中 | `0.08 + 0.04*sin(t*8)` | 0.04~0.12 |
| 隐身显形 | `0.08 + 0.92*(fadeIn/0.5)` | 0.08→1.0 (0.5s) |

## LOD 分级

| 条件 | 等级 | 效果 |
|------|------|------|
| lowFPS 或怪>50 | 4 | 跳过状态特效 |
| 怪>35 | 3 | 最低特效 |
| 怪>18 | 2 | 减半特效 |
| 其他 | 1 | 全特效 |

渲染上限：lowFPS 或怪>60 时只绘制前 50 个（按 Y 排序）。
