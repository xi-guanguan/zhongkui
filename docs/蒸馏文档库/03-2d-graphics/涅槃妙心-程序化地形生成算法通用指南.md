## 四大核心算法

### 1. 随机漫步（Random Walk）

适合生成**洞穴、隧道、河流**。

```lua
local x, y = startX, startY
for i = 1, steps do
    map[y][x] = FLOOR
    local dir = dirs[math.random(#dirs)]
    x = x + dir.x
    y = y + dir.y
end
```

**控制参数**：步数、方向权重（偏好水平/垂直）、分支概率。

### 2. 柏林噪声（Perlin Noise）

适合生成**自然地貌、高度图、生物群落**。

```lua
for y = 1, H do
    for x = 1, W do
        local val = perlin(x * scale, y * scale)
        if val > 0.6 then map[y][x] = MOUNTAIN
        elseif val > 0.3 then map[y][x] = FOREST
        else map[y][x] = PLAIN end
    end
end
```

**技巧**：多层叠加（octave），低频定大势，高频加细节。

### 3. 细胞自动机（Cellular Automata）

适合生成**自然洞穴、岛屿、有机形状**。

```lua
-- 初始化：随机填充 45% 墙壁
-- 迭代规则：周围8格中墙壁≥5则变墙，否则变空
for iter = 1, 5 do
    for y, x in cells do
        local walls = countNeighborWalls(x, y)
        newMap[y][x] = (walls >= 5) and WALL or FLOOR
    end
end
```

**迭代次数**：4-6 次收敛，次数越多洞穴越光滑。

### 4. BSP 树（Binary Space Partition）

适合生成**房间+走廊结构（地牢）**。

```
空间 → 递归二分 → 叶节点生成房间 → 兄弟节点连走廊
```

## Flood Fill 连通性检查

生成后必须验证所有可达区域连通：

```lua
local visited = floodFill(map, startX, startY)
-- 找到未访问的地板 → 挖通道连接 或 填充为墙
```

## Auto-Tile（自动贴图）

根据相邻格子自动选择正确的瓦片变体：

```lua
-- 4位掩码（上右下左是否同类型）
local mask = 0
if map[y-1][x] == WALL then mask = mask | 0x1 end  -- 上
if map[y][x+1] == WALL then mask = mask | 0x2 end  -- 右
if map[y+1][x] == WALL then mask = mask | 0x4 end  -- 下
if map[y][x-1] == WALL then mask = mask | 0x8 end  -- 左
local tileId = autoTileSheet[mask]  -- 16种变体
```

扩展到8方向 = 47种（使用 marching squares 简化为 16 种）。

## 可破坏地形

```lua
function DestroyTerrain(x, y, radius)
    for dy = -radius, radius do
        for dx = -radius, radius do
            if dx*dx + dy*dy <= radius*radius then
                map[y+dy][x+dx] = AIR
                dirtyChunks[getChunk(x+dx, y+dy)] = true
            end
        end
    end
    -- 只重建脏 chunk 的网格
    rebuildDirtyChunks()
end
```

## 性能优化

| 策略 | 说明 |
|------|------|
| 分块加载 | 只生成/渲染视口附近的 chunk |
| 对象池 | 瓦片精灵复用，不反复创建销毁 |
| 脏标记渲染 | 只重绘变化的 chunk |
| 延迟生成 | 玩家接近时才生成远处区块 |

## 完整模板：矿洞地图

```
1. 细胞自动机生成基础洞穴
2. Flood Fill 确保连通
3. 随机漫步挖矿脉（特殊方块）
4. 入口/出口放在最远两点
5. Auto-Tile 美化边缘
```

## 完整模板：地牢

```
1. BSP 分割空间生成房间
2. 兄弟节点最近点连走廊
3. 细胞自动机平滑走廊边缘
4. 随机放置宝箱/怪物/陷阱
5. 入口第一个房间，Boss最后一个
```
