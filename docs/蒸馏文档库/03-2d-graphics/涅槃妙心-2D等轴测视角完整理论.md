# 2D 等轴测视角：完整理论

> **作者**: 涅槃妙心 | **级别**: L2-参考 | **标签**: `等轴测` `Isometric` `投影` `绘制顺序` `建筑`

## 基础概念

等轴测（Isometric）使用 2:1 宽高比菱形格子，模拟 3D 俯视效果：

```
    /\
   /  \     宽:高 = 2:1
  /    \    常用: 64×32, 128×64
  \    /
   \  /
    \/
```

---

## 投影公式

### 格子坐标 → 屏幕坐标

```lua
local screenX = (col - row) * TILE_W / 2
local screenY = (col + row) * TILE_H / 2
```

### 屏幕坐标 → 格子坐标（逆变换）

```lua
local col = math.floor((screenX / (TILE_W/2) + screenY / (TILE_H/2)) / 2)
local row = math.floor((screenY / (TILE_H/2) - screenX / (TILE_W/2)) / 2)
```

## 地图分层

| 层级 | 内容 | 绘制顺序 |
|------|------|----------|
| 0 | 地面（grass/water/road） | 最先 |
| 1 | 地面装饰（花/石子） | — |
| 2 | 建筑/墙体 | 按 col+row 排序 |
| 3 | 角色/NPC | 按 col+row 排序 |
| 4 | 顶层特效/UI | 最后 |

## 道路系统

用参数化定义道路连接方向：

```lua
-- 4位掩码: 上右下左 (NESW)
-- 0b1010 = 上+下 = 直道
-- 0b1100 = 上+右 = 弯道
local roadTile = roadSpriteSheet[mask]
```

## 区域系统

用 floodFill 划分区域（住宅区/商业区/工业区），每个格子存区域 ID：

```lua
map[row][col] = { terrain = "grass", zone = "residential" }
```

## 建筑构造流程

```
1. 地面菱形（底面轮廓）
2. 向上挤出墙体（高度 = 楼层 × 层高）
3. 左面加阴影（模拟光照）
4. 顶面加屋顶（平顶/尖顶/斜顶）
5. 叠加装饰（窗户/门/阳台）
6. 地面投射阴影
```

## 绘制顺序算法

**核心规则**：`sortKey = col + row`（越大越后画）

```lua
table.sort(objects, function(a, b)
    local keyA = a.col + a.row
    local keyB = b.col + b.row
    if keyA == keyB then
        return a.layer < b.layer  -- 同位置按层级
    end
    return keyA < keyB
end)
```

**为什么有效**：等轴测中，col+row 相同的格子在同一"深度线"上，值越大的越靠近"屏幕前方"。

---

## 完整渲染管线

```
1. 计算可见范围（基于相机位置 + 视口大小）
2. 裁剪不可见格子
3. 逐层绘制：地面 → 建筑/角色（排序后） → 特效
4. UI 叠加（不参与排序）
```

---

## 实用建议

- 地图编辑器中用 **世界坐标**（col, row），渲染时才转屏幕坐标
- 大地图用 **分块加载**，只渲染视口 ± 1 chunk 的内容
- 角色移动时实时更新 sortKey，不要只在格子切换时更新
