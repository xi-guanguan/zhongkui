## UI 组件 Prompt 通用模板

### 按钮
```
[风格词] game button design, [状态: normal/hover/pressed/disabled],
[形状: rounded rectangle/circle/diamond],
[装饰: icon left/text center/badge corner]
```

### 面板/窗口
```
[风格词] game panel/window design, [尺寸描述],
[边框风格], [背景处理], title bar with close button,
content area with [布局描述]
```

### 图标集
```
[风格词] game icon set, consistent [尺寸] grid,
[主题: items/skills/buffs/currencies],
clear silhouette on [背景], [数量] icons in grid layout
```

### HUD 叠加
```
[风格词] game HUD overlay design, screen corners layout,
health/mana bars [位置], minimap [位置],
skill hotbar [位置], semi-transparent on gameplay background
```

## 实用技巧

1. **保持一致性**: 一个项目只用一种 UI 风格基底
2. **组件拆分**: 按钮、面板、图标分别生成，保证模块化
3. **状态设计**: 每个交互元素生成 normal/hover/pressed/disabled 4 态
4. **适配尺寸**: 移动端按钮最小 44×44dp，PC 端可更精细
5. **配色提取**: 从生成的 UI 图中提取主色调建立调色板
6. **九宫格切图**: 面板/按钮按九宫格切图规则设计，方便缩放复用
