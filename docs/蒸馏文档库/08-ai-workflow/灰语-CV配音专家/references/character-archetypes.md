# 角色原型模板库 — Character Voice Archetypes

> 30+ 即用型角色声音模板。选择最接近的原型，按需微调。

---

## 使用方法

1. 找到最接近目标角色的原型
2. 复制 `voice_prompt`（可微调关键词）
3. 复制 `audition_line`（或用角色自己的台词替换，须 >= 100 字符）
4. 调用 `audition_voices_for_character`
