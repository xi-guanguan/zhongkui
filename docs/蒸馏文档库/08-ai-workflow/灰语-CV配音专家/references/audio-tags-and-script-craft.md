## 一、Audio Tags 完整列表

**用法**: 在台词文本中插入英文方括号标签，控制语气和情绪。

| `[enthusiastic]` | 热情洋溢 | `[enthusiastic] 这将是一场伟大的冒险！` |

## 二、标点符号控制节奏

**v3 模型不支持 SSML `<break>` 标签**，必须用标点和 Audio Tags 控制节奏。

| 标点 | 效果 | 示例 |
|------|------|------|
| `，` / `,` | 短暂停顿 | `走吧，我们出发。` |
| `。` / `.` | 正常停顿 | `一切准备就绪。` |
| `……` / `...` | 思考/犹豫停顿 | `也许……我错了。` |
| `——` / `—` | 语气转折/中断 | `我以为——算了，不重要。` |
| `！` / `\!` | 强调语气 | `不可能！` |
| `？` / `?` | 疑问上扬 | `你确定？` |
| `？！` / `?\!` | 震惊反问 | `你说什么？！` |

**技巧**: 叠加省略号增加停顿时长：
- `也许...我错了` → 短停顿
- `也许......我错了` → 长停顿

## 三、台词编写黄金法则

### 法则 1: 匹配声音人设
台词风格必须与角色声音设定一致。温柔角色用温柔台词，暴躁角色用暴躁台词。

```
❌ 声音设定: gentle, warm → 台词: "给我滚！"
✅ 声音设定: gentle, warm → 台词: "[softly] 没关系的，慢慢来。"
```

### 法则 2: 用口语，不用书面语
TTS 读书面语会显得机械僵硬。

```
❌ "鉴于当前局势，我认为我们应当撤退。"
✅ "情况不妙啊，咱们先撤吧。"
```

### 法则 3: 加入语气词
适当加入"嗯、啊、呢、吧、呀、哎"等语气词增加自然感。

```
❌ "你来了。"
✅ "嗯，你来了呀。"
```

### 法则 4: 用标点控制呼吸
长句必须用逗号分割，否则 TTS 会连读到喘不过气。

```
❌ "在那遥远的山巅之上有一座被遗忘的古老神殿据说里面藏着上古的宝藏"
✅ "在那遥远的山巅之上，有一座被遗忘的古老神殿。据说里面……藏着上古的宝藏。"
```

### 法则 5: 试音台词的黄金长度
- **最少 100 字符**（API 硬性要求）
- **理想长度**: 150-250 字符
- **包含**: 至少 2 种情绪变化或语气转折

## 四、场景模板速查

### 战斗场景
```lua
"[shouting] 全体注意！敌人从右翼包抄过来了！[pause] 弓箭手，准备射击！[furious] 给我放箭！"
```

### 日常对话
```lua
"嗯……今天天气不错呢。[chuckling] 你看那边，小猫又在追蝴蝶了。"
```

### 悲伤告别
```lua
"[sad] 我知道……这一天终究会来。[sighs] 保重吧，朋友。[softly] 希望我们还能再见。"
```

### 搞笑场景
```lua
"[excited] 哈！看我新研发的超级无敌——[pause] 等等，这个按钮是干什么的？[gasps] 不不不不——！"
```

### 神秘旁白
```lua
"[dramatic tone] 在时间的尽头……有一扇门。[pause] 没有人知道门后是什么。[whispering] 也没有人……敢去打开它。"
```

### Boss 登场
```lua
"[laughing] 终于来了吗？[pause] 我等你们……很久了。[dramatic tone] 让我看看，所谓的勇者……究竟有几分实力。"
```
