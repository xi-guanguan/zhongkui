# 引导问诊流程 — Guided Voice Consultation

> 用户不懂配音术语，但知道什么好听。用选择题代替开放题，用比喻代替术语。

---

## 核心原则

1. **禁止术语轰炸**: 不说"音色质感"，说"声音听起来像什么"
2. **选择题优先**: 给 2-4 个选项，不问"你想要什么样的声音"
3. **用比喻代替描述**: "像大叔还是像少年"比"low-pitched or high-pitched"好懂
4. **先确认再生成**: 生成前必须让用户确认声音方向和台词内容
5. **快速迭代**: 先出 3 个候选声音让用户听，再根据反馈微调

---

## 五阶段问诊流程

### 阶段 1: 角色基础信息

**目标**: 确定性别和年龄段。

**问法**:
```
这个角色是男性还是女性？（或者是没有明确性别的角色，比如精灵、机器人？）

年龄大概是：
A. 少年/少女（十几二十岁，青春活力）
B. 青年（二十多三十岁，成熟但年轻）
C. 中年（四五十岁，沉稳有阅历）
D. 老年（六七十岁以上，沧桑智慧）
```

**如果用户已有角色设定文档**: 直接阅读文档提取信息，跳过本阶段。

---

### 阶段 2: 声音感觉

**目标**: 确定音色方向（温暖/清冷/沙哑/柔软等）。

**问法（用比喻）**:
```
你希望这个角色的声音听起来像：
A. 温暖的壁炉 — 让人安心、舒服（温暖厚实型）
B. 清晨的山泉 — 干净、清亮、通透（清亮通透型）
C. 砂纸打磨 — 沙哑、粗犷、有故事感（沙哑粗犷型）
D. 深夜电台DJ — 有磁性、性感、迷人（磁性诱人型）
E. 风吹丝绸 — 轻柔、温柔、像在耳边低语（柔软轻盈型）
F. 远处的雷声 — 低沉、浑厚、有压迫感（低沉浑厚型）
```

**追问（如果用户犹豫）**:
```
想象一下这个角色跟你说话，你的第一感觉是：
- 让人放松想靠近？（温暖/柔软）
- 让人精神一振？（清亮/磁性）
- 让人有点紧张敬畏？（沙哑/低沉）
```

---

### 阶段 3: 性格气质

**目标**: 确定情绪基调和说话风格。

**问法**:
```
这个角色的性格最接近哪一种：
A. 自信满满，天生领袖（自信权威型）
B. 温柔体贴，让人安心（温柔亲和型）
C. 高冷疏离，不太爱说话（冷漠疏离型）
D. 活泼开朗，像个小太阳（活泼开朗型）
E. 神秘莫测，让人捉摸不透（神秘深邃型）
F. 狡猾奸诈，笑里藏刀（阴险狡诈型）
G. 热血沸腾，永远不认输（热血激昂型）
H. 慵懒散漫，啥都无所谓（慵懒散漫型）
```

**说话速度**:
```
这个角色说话的速度：
A. 快——像机关枪一样停不下来
B. 正常——不快不慢
C. 慢——一字一句，很有份量
```

---

### 阶段 4: 确认方案

**目标**: 汇总信息，让用户确认。

**确认模板**:
```
好的，我来确认一下：

角色：[角色名]
性别：[男/女/中性]
年龄感：[少年/青年/中年/老年]
声音感觉：[温暖厚实 / 清亮通透 / ...]
性格气质：[自信权威 / 温柔亲和 / ...]
说话速度：[快/中/慢]

我会基于这些生成 3 个候选声音，你听一下选最接近的那个。
确认无误吗？有什么要调整的？
```

---

### 阶段 5: 准备试音台词

**目标**: 确认用于试音的台词。

**规则**:
1. 台词必须 >= 100 字符
2. 台词必须匹配角色性格和情绪
3. 优先用角色自己的游戏台词
4. 如果没有现成台词，从原型模板库选或现场编写

**问法**:
```
需要一句试音台词来生成声音。你有这个角色的台词吗？

如果没有，我可以根据角色性格写一句。比如：
"[根据角色类型写一句示例台词]"
这样的感觉可以吗？
```

## 反馈调优话术

**用户听完候选声音后**:

```
你觉得这几个声音怎么样？

A. 1号最接近，就用这个
B. 2号不错，但希望再 [温柔一点/低沉一点/快一点/...]
C. 都不太对，我想要的是 [描述]
D. 方向完全错了，我们重新来
```

**根据反馈调整**:

| 用户反馈 | 调整方向 |
|---------|---------|
| "声音太老了" | 降低年龄描述 |
| "不够有气势" | 加 commanding, authoritative |
| "太冷了，想要温暖一点" | 加 warm, gentle |
| "太假了 / 不自然" | 换 naturalistic, conversational 风格 |
| "太夸张了" | 去掉 over-the-top, anime style |
| "声音太高了" | 加 deep, low-pitched |
| "缺少感情" | 降低 stability 参数（0.3-0.5） |
