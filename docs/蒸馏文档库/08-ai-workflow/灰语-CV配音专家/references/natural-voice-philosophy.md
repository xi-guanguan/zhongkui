# 自然声音核心理念 — Anti-AI Voice Philosophy

> 最高优先级原则：生成的声音必须听起来像真人，不像 AI。
> 本文档是所有配音工作的底层信仰，贯穿声音设计、台词编写、参数调优全流程。

---

## 第零法则

**如果声音听起来像 AI，那一切都白做了。**

技术再高、参数再准、原型再对——用户听到"AI味"的那一刻，沉浸感就碎了。
自然感不是锦上添花，是生死线。

---

## 一、AI 味的七宗罪

认识敌人，才能消灭敌人。以下是 AI 声音最常见的"假"的来源：

| # | 症状 | 听感描述 | 根本原因 |
|---|------|---------|---------|
| 1 | **情绪平板** | 全程一个语调，像在念稿 | stability 太高 / 台词缺乏情绪起伏 |
| 2 | **节奏机械** | 每个字之间间隔均匀，像节拍器 | 台词缺少标点变化，没有长短句交替 |
| 3 | **呼吸消失** | 长句一口气说完，不像人类需要换气 | 台词没有用逗号/句号分割自然呼吸点 |
| 4 | **情绪断裂** | 前一句开心后一句突然愤怒，没有过渡 | Audio Tags 切换太突兀，缺少渐变 |
| 5 | **语气词缺失** | 每句话都是完整书面语，没有"嗯""啊""呢" | 台词写得太"正确"，缺少口语化处理 |
| 6 | **重音错位** | 该强调的没强调，不该强调的反而重读 | 台词结构不清晰，关键词没有用标点隔开 |
| 7 | **千人一面** | 不同角色听起来差别不大 | prompt 只写了年龄性别，缺少个性化维度 |

## 二、反 AI 味的核心武器

### 武器 1: 不完美才是真实

真人说话有犹豫、有重复、有语气词填充。刻意加入"不完美"让声音更真。

```
❌ AI 味（完美书面语）:
"我认为我们应该立即出发前往北方的雪山。"

✅ 真人感（口语化 + 犹豫）:
"我觉得吧……咱们还是赶紧出发吧，去北边那个——雪山。"
```

**技巧清单**:
- 加语气词：嗯、啊、呢、吧、呀、哎、哦、嘛、喂
- 加犹豫：用省略号 `……` 表示思考
- 加口头禅：每个角色可以有自己的口头禅（"话说回来""说实话""你看啊"）
- 加自我纠正：`"去那边的——不对，去这边！"`

---

### 武器 2: 呼吸节奏

人类说话需要呼吸。没有呼吸感的语音一听就是机器。

**呼吸点插入规则**:

| 句子长度 | 处理方式 |
|---------|---------|
| < 10 字 | 不需要逗号，一口气说完 |
| 10-20 字 | 中间至少 1 个逗号 |
| 20-30 字 | 至少 2 个逗号或 1 个句号分割 |
| > 30 字 | 必须拆成多个短句 |

```
❌ 窒息感（没有呼吸点）:
"在那遥远的山巅之上有一座被遗忘的古老神殿据说里面藏着上古的宝藏"

✅ 自然呼吸:
"在那遥远的山巅之上，有一座被遗忘的古老神殿。据说里面……藏着上古的宝藏。"
```

---

### 武器 3: 长短句交替

真人说话不会每句话都一样长。长短交替制造自然韵律。

```
❌ 机械等长:
"我们需要找到那把钥匙。然后打开那扇门。最后消灭最终的敌人。"

✅ 长短交替:
"钥匙。我们得先找到那把钥匙——然后打开大门，冲进去。[pause] 至于最后那个家伙……到时候再说。"
```

**节奏公式**: 短 → 长 → 短 → 长，像呼吸一样有节奏。

---

### 武器 4: 情绪渐变而非跳变

真人的情绪是渐变的，不是开关式切换。

```
❌ 情绪跳变（不自然）:
"[laughing] 哈哈哈太好笑了！[angry] 你给我滚！"

✅ 情绪渐变（自然）:
"[laughing] 哈哈哈太好笑了！[pause] 等等……[serious tone] 你刚才说什么？[angry] 你再说一遍？！"
```

**渐变路径**:
- 开心 → 疑惑 → 愤怒（不是 开心 → 愤怒）
- 平静 → 紧张 → 恐惧（不是 平静 → 恐惧）
- 悲伤 → 接受 → 释然（不是 悲伤 → 开心）

---

### 武器 5: Stability 是灵魂参数

**stability 越低，声音越"活"**。这是对抗 AI 味最直接的参数。

| stability 值 | 效果 | 适用 |
|-------------|------|------|
| 0.2-0.4 | 情感丰富，每次生成略有不同，最像真人 | 戏剧性强的台词、情感爆发 |
| 0.4-0.6 | 平衡区间，有变化但不会失控 | 大多数日常对话 |
| 0.6-0.8 | 稳定一致，适合正式场合 | 旁白、教程、系统提示 |
| 0.8-1.0 | 非常稳定，几乎没有变化 | 仅限机器人/AI 角色（刻意的 AI 感） |

**黄金法则**: 默认从 0.45 开始，不要从 0.5 开始。宁可稍微不稳定，也不要听起来像机器。

---

### 武器 6: 角色专属语言习惯

每个人说话都有独特的习惯。给角色设计专属的语言特征，一开口就能分辨出是谁。

**设计维度**:

| 维度 | 示例 |
|------|------|
| 口头禅 | "话说回来""你懂的""切""嘛" |
| 句尾习惯 | 总是用问句结尾、总是用感叹号、总是拖长尾音 |
| 称呼方式 | "小子""笨蛋""阁下""老弟" |
| 断句习惯 | 喜欢短句、喜欢长句、经常说到一半停下来 |
| 用词偏好 | 文雅角色用成语、粗犷角色用俚语、年轻人用网络用语 |

```
-- 同一句话，不同角色的自然版本：

-- 智者（缓慢、文雅）:
"嗯……如此说来，你已经做好了觉悟。好，那便去吧。"

-- 热血少年（急促、直接）:
"什么？你要去？那还等什么！走啊！冲！"

-- 冷酷剑客（简短、停顿多）:
"去吧。[pause] 别回头。"

-- 搞笑NPC（碎嘴、自我纠正）:
"你要走了？哎哟那个——路上小心啊！带把伞！不对，带把剑！算了都带上！"
```

## 三、台词自检清单

每条台词生成前，对照检查：

- [ ] 有没有语气词？（嗯/啊/呢/吧/呀）
- [ ] 长句有没有逗号分割呼吸点？
- [ ] 有没有长短句交替？
- [ ] 情绪变化是渐变还是跳变？
- [ ] 用的是口语还是书面语？
- [ ] 这个角色说这句话自然吗？符合他的性格吗？
- [ ] stability 是不是从 0.45 而不是 0.5 开始？

**任何一项不通过，先修改台词再生成。**

---

## 四、Voice Prompt 的自然感加持

在声音设计的 prompt 中，加入以下关键词可以增强自然感：

**推荐加入**:
```
naturalistic, conversational         -- 自然对话感（最重要）
with natural breathing pauses        -- 自然呼吸停顿
organic delivery                     -- 有机的表达方式
lifelike                             -- 逼真的
human-like warmth                    -- 人类般的温度
```

**避免使用**:
```
perfect diction                      -- 完美咬字 → 太机械
robotic                              -- 机器人感 → 除非角色就是机器人
monotone                             -- 单调 → 除非角色设定如此
synthetic                            -- 合成感 → 永远不要
flat delivery                        -- 平板表达 → 永远不要
```

## 五、终极原则

> **写台词的时候，念出来。**
>
> 你自己念一遍这句台词。
> 念的时候自然地在哪里停顿了？加逗号。
> 念的时候自然地加了什么语气词？写进去。
> 念的时候觉得别扭？那就改到不别扭为止。
>
> 如果你自己念着都像机器人，AI 生成出来只会更像。
