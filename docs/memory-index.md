# 记忆索引

> 项目详细上下文。每次 POST 更新。

## 项目概览

- **项目**: 黑笑话钟馗 — 腾讯云黑客松 AI CAN DO IT（赛题二：文化表达类）
- **本工作区角色**: 策划区（只写 `docs/`，不碰 `src/assets/`）
- **实现侧**: CodeBuddy 在 GitHub 仓库完成
- **仓库**: https://github.com/xi-guanguan/zhongkui.git

## 核心设计

- **玩法**: 钟馗用锁链钩鬼（原型：千禧小卖部套牛 → 换皮转换）
- **5 鬼阵容**: 牛头、马面、黑无常、白无常、刑天
- **特色**: 每抓到一只鬼弹出黑笑话
- **技术栈**: HTML5 Canvas (320×480)，纯 ES5+ JavaScript
