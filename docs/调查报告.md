### 核心代码分析：src/index.html (1196行)

#### 技术栈

| 技术 | 细节 |
|------|------|
| 画布 | HTML5 Canvas, 320×480 固定分辨率 |
| 语言 | 纯 ES5+ JavaScript, 无框架无依赖 |

```

│
├─ 音效系统 (L280-320)
│   └─ SFX: 9种程序化音效 (coin/hit/lasso/catchOk/escape/land/bigwin/tick/btn)
│
├─ 渲染模块 (L320-650)
│   ├─ cow(): 像素牛渲染（含表情系统: normal/struggle/dizzy/taunt）
```

#### 游戏流程（8阶段状态机）

```
IDLE（待机）
  ↓ 投币 + 按开始
RUNNING（牛群跑过）
  ├─ OBSERVE: 8秒观察期
  └─ COUNTDOWN: 5秒倒计时
  ↓ 点击/空格 抛绳
LASSO（丢绳动画 0.5秒）
  ↓
HITTING（拍打阶段 6秒）
  ↓ 时间到
RESULT（结果展示 3.5秒）
  ↓
SETTLE（结算）
  ↓ 点击继续
IDLE（回到待机）
```

#### 特色系统

1. **爆奖控制** — 最近10局收益率 > 80% 压低高赔率，< 40% 拉高低赔率
2. **体力系统** — 影响拉力速度 (100→85%, 80→55%, 50→30%, 20→10%)
3. **道具加成** — 玩具叠加拍打力度 (+20%~+80%)
4. **视觉反馈** — 屏幕震动/粒子/Big Win/残影/挤压拉伸/断绳碎片
