## 1. 打工/黄金矿工 — 30秒铜钱抓取

### 设计定位

**这是一个无聊的赚钱小游戏**，类似"刮个爽"里的擦盘子。
目的：给输光的玩家一个回血渠道，花时间换铜钱。

### 核心规则

| 参数 | 值 | 说明 |
|------|-----|------|
| 单局时长 | **30秒** | 固定，无延长机制 |
| 物品种类 | **仅铜钱** | 大铜钱 + 小铜钱，无其他道具 |
| 入场费 | 0 | 无门槛 |
| 难度曲线 | **无** | 每局铜钱随机分布，不递增难度 |
| 重复关卡 | **无** | 没有"第几关"概念，每局独立随机 |

### 操作机制

```
┌─────────────────────────┐
│  ⏱ 28s        💰 12铜   │  ← HUD: 倒计时 + 本局已抓
│                          │
│     🪙    🪙             │  ← 铜钱散布区域
│        🪙      🪙        │
│  🪙         🪙    🪙     │
│      🪙   🪙        🪙   │
│                          │
│         ╲│╱             │  ← 钩子（从顶部垂下）
│          │               │
│    ┌─────┼─────┐        │  ← 钟馗/船（顶部固定）
└─────────────────────────┘
```

**操作流程**：
1. 钩子自动左右摆动（匀速）
2. 玩家**点击屏幕** → 钩子向当前方向射出
3. 钩子碰到铜钱 → 抓住 → 自动收回（收回速度固定）
4. 钩子没碰到东西 → 到达底部/最大距离 → 自动收回
5. 收回后继续摆动，等待下一次点击
6. 30秒结束 → 结算

### 铜钱生成规则

| 类型 | 尺寸 | 价值 | 生成概率 | 视觉 |
|------|------|------|---------|------|
| 小铜钱 | 1× | 1 单位 | 70% | 标准铜钱 |
| 大铜钱 | 1.5~2× | 3 单位 | 30% | 放大 + 轻微旋转 |

**程序化布局**（每局随机重生）：
```javascript
// 伪代码
function generateCoins() {
  var coins = [];
  var count = randomInt(8, 12);  // 每局 8~12 个铜钱
  for (var i = 0; i < count; i++) {
    coins.push({
      x: random(50, SCREEN_W - 50),
      y: random(HOOK_START_Y + 60, SCREEN_H - 40),
      size: Math.random() < 0.3 ? 'big' : 'small',
      rotation: random(0, Math.PI * 2)  // 随机旋转角
    });
  }
  return coins;
}
```

### 收入结算

```
本局铜钱价值总和 × 好感系数 = 实际获得铜钱
```

好感系数（来自经济文档§2）：

| 好感等级 | 固定收入 | 说明 |
|---------|---------|------|
| Lv.1 | 5 铜钱/局 | 初始 |
| Lv.2 | 10 铜钱/局 | |
| Lv.3 | 20 铜钱/局 | |
| Lv.4 | 35 铜钱/局 | |
| Lv.5 | 50 铜钱/局 | 满级 |

> **注意**：收入是**固定值**（由好感等级决定），不是"抓到几个铜钱就得几个"。
> 抓铜钱的过程只是动画表演（类似主玩法的虚假斯金纳箱思路）。
> 只要玩家花了30秒，就给固定收入。铜钱抓取量只影响视觉反馈感。

### 钩子参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 摆动角度 | ±60° | 从中心向左右各60° |
| 摆动周期 | 2秒/个往返 | 不快不慢 |
| 射出速度 | 400px/s | 快速射出 |
| 收回速度 | 300px/s | 稍慢（空钩）/ 200px/s（带铜钱） |
| 钩子长度 | 屏幕高度 80% | 能到底部 |

### 状态机

```
IDLE（摆动等待点击）
  → 点击 → SHOOTING（射出）
    → 碰到铜钱 → RETRACTING_WITH_COIN
    → 到底部 → RETRACTING_EMPTY
      → 回到顶部 → IDLE
```

### 不做的事（极简原则）

- ❌ 多种物品（只有铜钱）
- ❌ 障碍物/危险物
- ❌ 难度递增
- ❌ 关卡系统
- ❌ 连击奖励
- ❌ 特殊道具/加速/减速
- ❌ 分数排行

## 2. 孟婆对话 — 间断式看板娘

### 设计定位

**像二次元游戏的角色一样**，简单间断的对话。
不是剧情系统，只是增加"陪伴感"和"人格表达"。

### 触发时机

| 场景 | 触发 | 对话类型 |
|------|------|---------|
| 进入孟婆店 | 每次切换到店 | 随机寒暄（1句） |
| 购买奶茶 | 完成购买 | 推荐/感谢（1句） |
| 好感升级 | 等级提升瞬间 | 好感台词（1句） |
| 空闲3秒 | 在店内无操作 | 闲聊（1句） |
| 打工归来 | 打工结束回到店 | 迎接（1句） |

### 对话格式

```
┌─────────────────────────┐
│                          │
│    [孟婆立绘]            │
│                          │
│  ┌──────────────────┐   │
│  │ "今天想喝点什么？"  │   │  ← 气泡框，2~3秒后自动消失
│  └──────────────────┘   │
│                          │
│   [奶茶列表...]          │
└─────────────────────────┘
```

### 对话池（按场景分类，每次随机取1句）

#### 进店寒暄
```
"来啦？"
"今天气色不错嘛。"
"冥界天气真热，来杯冷的？"
"老样子？"
"又来找我了～"
```

#### 购买反馈
```
"好眼光。"
"这杯我也爱喝。"
"要加珍珠吗？哦已经有了。"
"祝你好运～"
```

#### 好感升级
```
"嗯...算你还不赖。"          -- Lv.2
"行吧，算你半个熟客。"        -- Lv.3
"每天都来，还挺有毅力。"      -- Lv.4
"...你是我最好的客人了。"     -- Lv.5
```

#### 空闲闲聊
```
"发什么呆呢？"
"要不...出去抓两只？"
"站着也不点单啊。"
```

#### 打工归来
```
"辛苦了，喝杯茶歇歇。"
"赚到钱了？让我看看。"
"又去钩铜钱了？"
```

### 实现要点

| 要点 | 说明 |
|------|------|
| **不阻塞** | 对话不暂停游戏，纯展示 |
| **自动消失** | 显示 2~3 秒后淡出 |
| **不重复** | 同一场景连续进入不重复上一句 |
| **可扩展** | 后期可接 AI 生成（腾讯云API），当前先写死 |
| **无分支** | 没有选择/回复，纯单向展示 |

### 技术实现（极简）

```javascript
// mengpo.js
var MengPo = {
  lines: {
    enter: ["来啦？", "今天气色不错嘛。", "老样子？", "又来找我了～"],
    buy:   ["好眼光。", "这杯我也爱喝。", "祝你好运～"],
    idle:  ["发什么呆呢？", "要不...出去抓两只？"],
    back:  ["辛苦了。", "赚到钱了？", "又去钩铜钱了？"],
    levelUp: { 2: "算你还不赖。", 3: "半个熟客了。", 4: "挺有毅力。", 5: "你是我最好的客人了。" }
  },
  lastLine: null,
  
  getLine: function(context) {
    var pool = this.lines[context];
    if (!pool) return null;
    // 不重复上一句
    var line;
    do { line = pool[Math.floor(Math.random() * pool.length)]; }
    while (line === this.lastLine && pool.length > 1);
    this.lastLine = line;
    return line;
  }
};
```

## 3. 数值验证

### 打工效率 vs 主玩法

| 好感等级 | 打工30s收入 | 主玩法30s（约3轮×1币）≈ EV |
|---------|------------|---------------------------|
| Lv.1 | 5 铜钱 | 3 × 0.458 ≈ 1.37 铜钱 |
| Lv.3 | 20 铜钱 | 同上 |
| Lv.5 | 50 铜钱 | 同上 |

> 打工始终正期望（有保底），主玩法始终负期望（赌博）。
> 对比清晰：花时间=稳定赚；花钱=赌运气。
> 初期打工收入低（5铜钱/30s），刻意无聊，推动玩家回去赌。
